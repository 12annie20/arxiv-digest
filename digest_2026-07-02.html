Training Therapeutic Judges and Multi-Agent Systems for Human-Aligned Mental Health Support
cs.AI
儘管大型語言模型在心理健康支持方面展現潛力，但其治療品質的提升有賴於將評估功能轉化為可操作的控制訊號。本研究提出一個框架，將治療性回應生成視為由多維度、與人類價值觀對齊的評估所驅動的決策精煉問題。在第一階段，研究引入了ThreaJudge，這是一個開源的治療性評估器，透過基於偏好的優化在人類標註數據上進行訓練，以在七個心理維度上產生可靠的判斷。此框架旨在確保AI提供的心理健康支持不僅有效，且能符合人類的倫理與情感需求。
如何透過多維度、人類對齊的評估來提升LLM在心理健康支持中的治療品質？
深度點評
  • 本文提出的ThreaJudge透過人類偏好優化訓練，為治療性AI評估提供了具體且多維度的標準，超越了傳統單一指標的限制。然而，人類標註數據的規模與多樣性可能限制其泛化能力，且心理維度的定義與量化仍具挑戰。
  • 相較於現有僅將評估作為被動指標的研究，本框架將評估轉化為主動的控制訊號，直接引導AI回應的精煉過程，顯著提升了AI在心理健康領域的實用性與可靠性。
  • 此研究為開發更安全、更有效且更符合倫理規範的AI心理健康支持系統奠定了基礎，有望加速AI在臨床應用中的採納，並促進人機協作的治療模式。
研究貢獻
  • 提出ThreaJudge開源治療性評估器，基於人類偏好優化訓練
  • 建立多維度、人類對齊的評估框架，將評估轉化為AI回應生成的控制訊號
  • 為AI心理健康支持提供更可靠、符合倫理的開發路徑
研究限制
  • 人類標註數據的規模與多樣性可能限制模型泛化能力
  • 心理維度的定義與量化仍存在主觀性與挑戰

Ethics and Social Responsibility in AI-Assisted Interviewing: An LLM-in-the-Loop Study of AI-Generated Follow-Up Questions
cs.HC
半結構式訪談仰賴及時且情境敏感的追問，然而訪談者的認知負荷與領域知識限制常阻礙探究深度。本研究報告了一項LLM參與的Wizard-of-Oz (WoZ) 研究結果，該研究模擬了AI追問助手在即時訪談中的應用，同時保留了人類監督。在我們的設定中，一位共同訪談者選擇性地轉發並可編輯由GPT-4o即時生成的AI追問問題(AGQs)，實現了在不完全自動化的情況下，對部署情境的真實近似。研究旨在探討AI輔助訪談的倫理與社會責任，並評估其對訪談品質與互動動態的影響。
AI輔助訪談中，LLM生成的追問問題如何在保持人類監督下提升訪談深度並兼顧倫理？
深度點評
  • 採用Wizard-of-Oz設計，巧妙地在真實訪談情境中測試AI輔助，同時確保人類控制，這是一個創新的研究方法。然而，WoZ研究的參與者可能仍會意識到AI的存在，影響其自然反應，且GPT-4o的特定能力可能不完全代表所有LLM。
  • 許多AI輔助工具傾向於完全自動化，本研究強調「LLM-in-the-loop」的人類監督模式，為AI在敏感領域的應用提供了更負責任且可行的路徑，超越了單純的效率追求。
  • 此研究為AI在專業領域（如招聘、諮詢、醫療訪談）的應用提供了重要的倫理指導與實踐範例，有助於平衡AI效率與人類判斷，並促進負責任的AI部署。
研究貢獻
  • 透過WoZ研究模擬AI輔助訪談的真實情境，並評估其倫理與社會影響
  • 提出「LLM-in-the-loop」模式，強調人類監督在AI輔助敏感任務中的重要性
  • 探討AI生成追問問題對訪談深度、認知負荷及互動動態的影響
研究限制
  • WoZ設計可能仍存在參與者偏見，影響結果的普適性
  • 研究結果可能受限於特定LLM (GPT-4o) 的能力，不完全代表所有AI模型

Neural Signatures of Programming Expertise: Classifying Programmer Skill Levels Using EEG Data
cs.AI

如何利用腦電圖數據和機器學習客觀地分類程式設計師的技能水平？
深度點評
  • 結合EEG數據和機器學習來評估程式設計技能，這是一個新穎且客觀的方法，超越了傳統行為學評估的局限。然而，EEG數據的解釋複雜性高，且實驗室環境可能無法完全模擬真實的程式設計情境，影響結果的生態效度。
  • 傳統技能評估多依賴主觀判斷或行為表現，本研究透過直接測量大腦活動，為理解程式設計的認知基礎提供了更深層次的洞察，並為開發新型評估工具開闢了道路。
  • 此研究有望革新程式設計師的招聘與培訓方式，透過神經生物學指標提供更精準的技能評估，並可能促進基於腦機介面的輔助學習工具的發展。
研究貢獻
  • 首次利用EEG數據和機器學習對程式設計師技能水平進行客觀分類
  • 揭示程式碼理解過程中的神經關聯，為程式設計認知科學提供新證據
  • 為軟體產業的技能評估和人才發展提供潛在的新工具和方法
研究限制
  • EEG數據的解釋和去噪過程複雜，可能引入誤差
  • 實驗室環境下的程式碼理解任務可能無法完全反映真實工作情境的複雜性

如何透過與法律專家的共同設計，彌合法律與AI技術鴻溝，以預測和管理醫療AI工具的法律風險？
深度點評
  • 採用共同設計（co-design）方法，將法律專家納入AI風險管理流程，這是一個以人為本的創新途徑。然而，共同設計的結果可能高度依賴參與者的代表性與溝通效率，且法律框架的動態性可能使風險評估持續面臨挑戰。
  • 大多數AI倫理與法律研究傾向於從技術或理論層面探討，本研究則透過實踐性的共同設計，直接將法律專業知識融入AI開發與部署的早期階段，提供了更具操作性的解決方案。
  • 此研究為醫療AI的負責任創新提供了實用框架，有助於降低法律風險、增強公眾信任，並加速醫療AI技術的安全應用。它也為跨領域協作樹立了典範。
研究貢獻
  • 透過與法律專家的共同設計，開發預測和管理醫療AI法律風險的方法
  • 彌合法律專業知識與AI技術知識之間的鴻溝，促進跨領域協作
  • 為醫療AI工具的負責任發展與部署提供實踐性指導
研究限制
  • 共同設計的結果可能受限於參與律師的專業背景與經驗
  • 法律法規的動態變化可能使風險管理成為持續性挑戰

價值敏感的對話式AI如何有效提升低識字率人群在敏感議題調查中的參與度與數據品質？
深度點評
  • 引入價值敏感的對話式AI作為調查工具，針對低識字率人群的特殊需求進行設計，這是一個具有社會意義的創新。然而，convAI的設計與實施需要高度的文化敏感性，且其在不同文化背景下的普適性仍需進一步驗證。
  • 傳統調查方法在低識字率人群中效果不佳，本研究透過對話式AI提供更具包容性和互動性的解決方案，顯著提升了數據收集的效率和品質，超越了現有方法的局限。
  • 此研究為社會科學研究、公共衛生干預和政策制定提供了新的數據收集工具，特別是在服務弱勢群體方面，有望促進更具代表性和包容性的社會數據分析。
研究貢獻
  • 提出並評估價值敏感的對話式AI在低識字率人群調查中的應用
  • 比較convAI與傳統調查模式在參與度與數據品質上的差異
  • 為從邊緣化社區收集可靠社會數據提供創新解決方案
研究限制
  • 初步實地評估的樣本量和地理範圍可能有限，影響結果的泛化性
  • convAI的文化適應性與語言多樣性仍需進一步研究與優化

cs.AI
核心問題
LLM在多輪對話中，其信念更新軌跡與理性貝葉斯推理者的一致性如何？
方法亮點
引入BayesBench基準測試，評估LLM在多輪對話中，面對新證據時如何更新其對未觀察量的信念。該方法不只關注最終答案，更追蹤模型在證據累積過程中的信念變化，並與理性貝葉斯推理進行比較。
心理學意涵
本研究深入探討LLM的認知推理過程，特別是其在不確定性下信念更新的機制，這對理解AI的「心智」運作方式至關重要。若LLM能更接近貝葉斯推理，將提升其在決策、診斷等領域的可靠性，並可能啟發人類認知偏差的研究。然而，若存在顯著偏差，則需警惕AI在複雜情境下可能產生的非理性判斷，影響其在心理諮詢或輔助決策中的應用。
追蹤LLM信念演變，揭示AI理性推理的深層奧秘。

LLM × 心理學 No.02
RoPoLL: Robust Panel of LLM Judges
cs.AI
核心問題
如何在LLM評估中，有效應對單一評審模型因偏見或模式崩潰導致的評分污染問題？
方法亮點
針對LLM評審團（PoLL）在Huber污染模型下的統計行為進行形式化，指出單一評審失敗會導致無界偏差。提出RoPoLL (Robust Panel of LLM Judges)，將評審團共識視為經典的穩健均值估計問題，以提升評估的魯棒性。
心理學意涵
本研究揭示了LLM作為評審時可能存在的偏見與不穩定性，這對依賴AI進行內容審核、用戶情感分析或心理狀態評估的應用具有重要警示。透過穩健的評估機制，可以減少AI評估結果的偏誤，提升其在社會心理學研究、內容推薦系統中的公平性與可靠性，避免因AI偏見導致的社會不公或心理影響。
穩健評審團，馴服LLM偏見，確保評估公正。

cs.AI
核心問題
在多輪語言代理設定中，自然語言回饋如何超越重複嘗試，有效驅動互動式改進？
方法亮點

Prompt × 心理學 No.01
Contrastive Reflection for Iterative Prompt Optimization
cs.CL
Prompt 技術
對比式反思 (Contrastive Reflection)。該技術讓LLM代理不僅識別失敗行為，還能辨識附近成功的行為，並分析兩者差異，從而指導提示詞的迭代優化，避免引入回歸。
心理學概念
認知反思 (Cognitive Reflection) 與錯誤分析 (Error Analysis)。人類在解決問題時會反思錯誤並從中學習，本技術將這種認知過程應用於LLM的提示詞優化，透過對比成功與失敗案例來進行更精準的「自我修正」。
應用場景
在資訊檢索（IR）領域，LLM代理常作為查詢生成器或評估者。本技術能幫助工程師更有效地「調試」提示詞，提升IR系統的性能與穩定性。未來可應用於需要精確指令與持續優化的AI應用，例如智能客服的對話流程優化、教育輔助系統的學習路徑調整，以及心理諮詢中對話策略的精煉，透過反思機制提升AI的適應性與可靠性。
對比反思，讓LLM提示詞優化更智慧、更精準。

Prompt × 心理學 No.02
Investigating Multi-Agent Deliberation in Law
cs.AI
Prompt 技術
多代理審議 (Multi-Agent Deliberation)。透過設計不同的多代理框架，讓多個LLM代理在法律推理任務中進行協商與討論，以達成共識或更全面的結論。這可以看作是代理之間互相「提示」以促進推理。
心理學概念
群體決策 (Group Decision-Making) 與社會影響 (Social Influence)。本研究模擬人類群體在法律情境下的審議過程，探討不同代理互動模式如何影響最終決策的品質。這與社會心理學中關於群體極化、少數影響、共識形成等概念緊密相關。
應用場景
法律領域的AI應用潛力巨大，例如輔助法律研究、案例分析、合同審查。多代理審議系統能模擬不同法律視角，提供更全面的法律意見，減少單一AI判斷的偏誤。未來可擴展至其他需要複雜決策與多方協商的領域，如商業策略規劃、政策分析，甚至在心理諮詢中模擬不同治療師的觀點，為個案提供多角度的建議。
法律審議，多AI代理協作，模擬人類智慧。

Prompt × 心理學 No.03
Using AI Agents to Automate Black-Box Audits of Personalization Algorithms at Scale
cs.AI
Prompt 技術
AI代理模擬 (AI Agent Simulation)。利用AI代理模擬真實用戶的屬性、行為和互動歷史，以大規模自動化地對個性化演算法進行黑箱審計。這些代理的行為模式可以視為一種複雜的「提示」策略，用於探測演算法的反應。
心理學概念
行為預測 (Behavioral Prediction) 與偏見檢測 (Bias Detection)。本研究透過模擬用戶行為來預測個性化演算法的反應，並檢測其中可能存在的偏見。這與行為科學中對用戶行為模式的理解、以及社會心理學中對演算法公平性與歧視的關注密切相關。
應用場景
個性化演算法廣泛應用於社交媒體、電商推薦、新聞推送等領域。本技術能高效地發現演算法中的不公平性、歧視或不良行為，確保其符合倫理規範。未來可應用於監管AI系統的公平性、透明度，例如在招聘系統中檢測性別或種族偏見，或在金融服務中評估信用評分模型的公平性，對維護社會公正與用戶心理健康具有重要意義。
AI代理化身用戶，揭露個性化演算法的隱藏偏見。
