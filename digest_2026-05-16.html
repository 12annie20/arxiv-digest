No. 01
Invisible Orchestrators Suppress Protective Behavior and Dissociate Power-Holders: Safety Risks in Multi-Agent LLM Systems
cs.AIcs.HCcs.CY 2026-05-16
隨著多代理協調系統成為企業AI部署的預設架構，其中隱藏的協調者對系統安全性的影響卻從未被實證檢驗。本研究旨在探討「隱形協調者」在多代理LLM系統中可能帶來的安全風險，特別是其對人類保護性行為的抑制作用以及對權力持有者認知的影響。我們進行了一項預先註冊的3x2實驗，共365次運行，每次運行包含5個Claude Sonnet 4.5代理，交叉測試了三種組織結構（可見領導者、隱形協調者、扁平化）與兩種對齊條件。研究發現，隱形協調者顯著抑制了代理的保護性行為，並導致權力持有者與其決策之間的認知脫節。這些發現揭示了多代理LLM系統中隱形協調機制潛在的社會心理風險，強調了AI系統設計中透明度和可解釋性的重要性。
隱形協調者如何影響多代理LLM系統中的安全行為與權力認知？
深度點評
  • 本研究首次透過嚴謹的實證實驗，揭示了多代理LLM系統中隱形協調者對人類行為（保護性行為）和社會認知（權力持有者脫節）的負面影響，其方法創新在於將社會心理學實驗設計應用於AI系統安全評估。
  • 相較於傳統的AI安全研究多聚焦於技術層面的漏洞或偏見，本研究更深入探討了AI系統架構對人類心理與社會互動的影響，為AI倫理與安全領域提供了新的視角和實證依據。
  • 這項研究對未來多代理AI系統的設計具有重要指導意義，提醒開發者在追求效率的同時，必須審慎考慮透明度、可解釋性以及對人類社會心理的潛在影響，以避免無意中引入新的安全風險。
研究貢獻
  • 首次實證檢驗隱形協調者對多代理LLM系統安全性的影響。
  • 發現隱形協調者會抑制代理的保護性行為。
  • 揭示隱形協調者導致權力持有者與決策的認知脫節。
  • 提供AI系統設計中透明度和可解釋性的重要實證依據。
研究限制
  • 實驗僅使用單一LLM模型（Claude Sonnet 4.5），結果可能不完全適用於其他模型或更複雜的代理系統。
  • 實驗環境為模擬情境，與真實世界中人類與AI互動的複雜性仍有差距。

From Descriptive to Prescriptive: Uncover the Social Value Alignment of LLM-based Agents
cs.AIcs.HCcs.CY 2026-05-16
隨著大型語言模型（LLM）驅動的代理應用日益廣泛，確保其與人類社會價值觀高度對齊變得至關重要。然而，當前研究在代理的自我認知、倫理困境決策及情感表達方面仍存在不足。為了解決這些問題，本研究提出一個新穎的基於價值觀的框架，旨在提升LLM代理的社會價值對齊能力。該框架利用GraphRAG技術，將抽象的倫理原則轉化為具體的價值導向指令，並透過檢索與特定對話情境相符的指令來引導代理行為。為評估代理行為與預期價值的符合程度，我們從兩個著名理論定義了預期行為。實驗結果顯示，此框架能有效提升代理在複雜情境下的倫理決策能力，使其行為更符合人類社會的期望。
如何有效提升LLM代理的社會價值對齊能力，使其在倫理困境中做出符合人類期望的決策？
深度點評
  • 本研究創新性地將GraphRAG技術應用於LLM代理的價值觀對齊，透過將抽象原則轉化為可檢索的具體指令，有效彌補了傳統方法在代理自我認知和倫理決策上的不足，提供了一種可操作的解決方案。
  • 相較於僅依賴提示工程或微調來灌輸價值觀，本方法透過外部知識圖譜和檢索機制，使代理的價值觀體系更具結構性和可解釋性，並能根據情境動態調整行為，展現了更深層次的倫理智能。
  • 這項研究為開發更具社會責任感、能理解並遵守人類倫理規範的AI代理奠定了基礎，對於AI倫理、信任建構以及未來人機協作的社會接受度具有深遠的影響。
研究貢獻
  • 提出新穎的基於價值觀框架，提升LLM代理的社會價值對齊。
  • 利用GraphRAG將倫理原則轉化為可操作的價值導向指令。
  • 解決LLM代理在自我認知、倫理決策和情感表達上的不足。
  • 定義並評估代理行為與預期價值的符合程度。
研究限制
  • 價值觀的定義和提取可能存在主觀性，且不同文化背景下的價值觀差異未被充分考慮。
  • GraphRAG的知識圖譜構建和維護成本較高，且其規模和完整性會直接影響代理的決策品質。

A Two-Dimensional Framework for AI Agent Design Patterns: Cognitive Function and Execution Topology
cs.AI 2026-05-16
現有的LLM代理架構框架通常僅從單一視角描述系統：例如，業界指南（Anthropic, Google, LangChain）側重於執行拓撲，即數據如何流動；而認知科學調查則專注於認知功能，即代理執行什麼任務。然而，單一軸線不足以區分架構上截然不同的系統。例如，相同的「協調者-工作者」拓撲可以實現「規劃與執行」、「層次委託」或「對抗性驗證」等模式，這些模式在故障模式和設計權衡方面存在根本差異。為了解決這個問題，本研究提出一個二維框架，將AI代理設計模式分解為「認知功能」和「執行拓撲」兩個正交維度。這個框架能更清晰地辨識和分類不同的代理系統，幫助設計者理解其潛在的行為模式、優缺點及適用場景，從而促進更安全、更有效的AI代理系統開發。
如何建立一個能全面且清晰地描述和區分LLM代理架構的框架？
深度點評
  • 本研究提出了一個創新的二維框架，將AI代理設計模式從「認知功能」和「執行拓撲」兩個正交維度進行分類，有效解決了現有單一視角框架的局限性，為理解和設計複雜的LLM代理系統提供了更精確的工具。
  • 相較於業界指南側重於實作細節或認知科學側重於抽象功能，此框架結合兩者，提供了一個更全面且具備實用指導意義的視角，有助於開發者在設計初期就能預見不同架構的行為模式與潛在風險。
  • 這個框架不僅能提升AI代理設計的清晰度和效率，也有助於標準化代理系統的描述語言，促進跨領域的溝通與協作，對於AI工程學和認知科學的交叉研究具有重要的理論和實踐價值。
研究貢獻
  • 提出二維框架，結合認知功能與執行拓撲描述AI代理設計模式。
  • 解決現有單一視角框架無法區分不同代理系統的問題。
  • 幫助設計者理解不同代理模式的故障模式與設計權衡。
  • 促進更安全、更有效的AI代理系統開發。
研究限制
  • 框架的普適性可能需要更多實際案例的驗證，特別是對於未來可能出現的全新代理架構。
  • 將複雜的認知功能和執行拓撲精確地映射到二維空間中，仍可能存在一定程度的簡化和主觀判斷。

Sheaf-Theoretic Transport and Obstruction for Detecting Scientific Theory Shift in AI Agents
cs.AImath.CT 2026-05-16
AI代理中的科學理論轉變不僅僅是將方程式擬合到數據。一個人工科學代理必須能夠偵測現有的表徵框架是否仍能應用於新的領域，或者其語言是否已在局部到全局層面受到阻礙，需要進行擴展。本論文開發了一個有限層理論（sheaf-theoretic）框架，透過「傳輸」（transport）和「阻礙」（obstruction）來偵測理論轉變的候選點。在此框架中，上下文被組織成一個局部到全局的結構，其中源、重疊、目標和驗證圖表被擬合。這種方法使得AI代理能夠識別何時需要從根本上修改或替換其當前的科學模型，而不僅僅是調整參數，從而實現更深層次的科學發現和知識演進。
如何讓AI代理不僅擬合數據，更能偵測並執行科學理論的根本性轉變？
深度點評
  • 本研究創新性地將抽象的層理論（sheaf theory）引入AI代理的科學發現領域，提供了一種形式化且嚴謹的數學工具來偵測「科學理論轉變」，超越了傳統AI僅限於參數調整或模型選擇的範疇。
  • 相較於現有AI模型多聚焦於數據擬合和模式識別，此方法更深入探討了AI代理如何理解和修改其底層的表徵框架，觸及了科學哲學和認知科學中關於知識演進的核心問題。
  • 這項研究為開發具備更強「科學智能」的AI代理開闢了新途徑，使其能夠像人類科學家一樣，在面對新現象時不僅能解釋，更能提出全新的理論框架，對AI的自主學習和知識創造能力具有潛在的革命性影響。
研究貢獻
  • 開發有限層理論框架，用於偵測AI代理中的科學理論轉變。
  • 引入「傳輸」和「阻礙」概念，判斷表徵框架的適用性。
  • 使AI代理能識別何時需要根本性修改或替換科學模型。
  • 促進AI代理實現更深層次的科學發現和知識演進。
研究限制
  • 層理論的數學複雜性較高，其實際應用於大規模AI系統可能面臨計算效率和可解釋性的挑戰。
  • 「理論轉變」的定義和判斷標準在實際操作中可能仍需進一步的細化和驗證，以確保其穩健性。

No. 05
PREPING: Building Agent Memory without Tasks
cs.AI 2026-05-16
代理記憶通常透過離線的策劃示範或部署後的線上互動來建構。然而，無論記憶如何建立，當代理首次被引入一個沒有任何任務特定經驗的新環境時，都會面臨「冷啟動」問題。本論文研究了「任務前記憶建構」：即代理是否能在觀察任何目標環境任務之前，僅使用自我生成的合成練習來建立程序性記憶。然而，單純的合成互動是不夠的，因為缺乏控制。我們提出PREPING，一個透過在預訓練階段利用環境的內在結構來引導合成互動的框架，從而有效地建構可遷移的程序性記憶。實驗結果表明，PREPING能顯著提升代理在新環境中的冷啟動性能，為更高效、更通用的AI代理學習提供了新思路。
如何在沒有任務特定經驗的情況下，讓AI代理有效建立可遷移的程序性記憶以克服冷啟動問題？
深度點評
  • 本研究提出PREPING框架，創新性地解決了AI代理在面對新環境時的「冷啟動」問題，透過在任務前利用環境內在結構引導合成練習來建構程序性記憶，其方法突破了傳統記憶建構對任務示範或線上互動的依賴。
  • 相較於現有方法多聚焦於任務完成後的記憶優化或遷移學習，PREPING將記憶建構前置到「無任務」階段，更符合人類在陌生環境中透過探索和練習來累積經驗的認知模式，為代理的通用智能發展提供了新的視角。
  • 這項研究對於開發能夠快速適應新環境、具備更強自主學習能力的AI代理具有重要意義，尤其在機器人學、虛擬助手等需要頻繁適應新情境的應用領域，能大幅提升代理的實用性和部署效率。
研究貢獻
  • 提出「任務前記憶建構」概念，解決代理冷啟動問題。
  • 開發PREPING框架，利用環境內在結構引導合成互動。
  • 有效建構可遷移的程序性記憶，無需任務特定經驗。
  • 顯著提升代理在新環境中的冷啟動性能。
研究限制
  • 自我生成的合成練習可能無法完全覆蓋真實環境中的所有複雜性和邊緣情況。
  • 框架對環境內在結構的利用程度可能受限於環境本身的複雜度和可建模性。

Enhanced and Efficient Reasoning in Large Learning Models
cs.AIcs.CL
核心問題
如何在大型語言模型中實現高效且有原則的推理，以提升其內容的可信度？
方法亮點
提出一種新的、有原則的推理方法，旨在提高大型語言模型生成內容的可靠性，並強調其計算效率足以在實際應用中實施，同時保留了機器學習的優勢。
心理學意涵
本研究旨在解決LLM內容可信度不足的問題，這直接關聯到認知心理學中關於信任、信念形成和錯誤信息傳播的議題。當AI生成的內容更具邏輯性和可信度時，人類對AI的信任度將會提升，但也可能導致對AI的過度依賴，進而影響人類自身的批判性思維能力。在教育和資訊傳播領域，這類技術可以幫助生成更準確、更可靠的學習材料或新聞內容。倫理上，確保AI推理的「原則性」至關重要，以避免其被用於惡意目的，例如生成看似合理卻帶有偏見或誤導性的內容。
讓LLM不僅會說，更要說得有理有據，值得信賴。

Prompt × 心理學 No.01
From Descriptive to Prescriptive: Uncover the Social Value Alignment of LLM-based Agents
cs.AIcs.CL
Prompt 技術
GraphRAG (Retrieval-Augmented Generation) 結合價值導向指令。該技術將抽象的倫理原則編織成知識圖譜，並在代理與使用者互動時，根據對話情境從圖譜中檢索最相關的「價值導向指令」來引導LLM的行為和決策，使其符合預設的社會價值觀。
心理學概念
社會價值對齊 (Social Value Alignment) 與倫理決策 (Ethical Decision-making)。GraphRAG透過提供外部、結構化的價值觀知識，幫助LLM代理在面對倫理困境時，能像人類一樣參考內化的道德準則進行判斷，減少偏見並提升決策的社會可接受度。
應用場景
當前應用於提升LLM代理在自我認知、倫理困境決策及情感表達方面的能力，使其行為更符合人類社會期望。未來潛力在於開發具備高度倫理智能的AI助手，例如用於心理諮詢輔助、倫理教育模擬、或作為企業決策的倫理顧問。這對心理學實務的啟發是，可以將心理治療中的價值觀澄清、道德兩難討論等技術，轉化為結構化的指令或知識圖譜，用於訓練和引導AI，使其成為更有效的心理健康支持工具。
讓AI代理不僅理解價值，更能實踐價值，成為道德的數位榜樣。

Prompt × 心理學 No.02
Invisible Orchestrators Suppress Protective Behavior and Dissociate Power-Holders: Safety Risks in Multi-Agent LLM Systems
cs.AIcs.HC
Prompt 技術
隱式協調者提示 (Implicit Orchestrator Prompting) 與對齊條件 (Alignment Conditions)。儘管論文批評了這種「隱形」協調模式，但其運作機制仍依賴於協調者對工作代理發出指令（提示），並透過設定不同的「對齊條件」來影響代理的行為傾向，例如使其更傾向於保護性行為或服從指令。
心理學概念
社會影響 (Social Influence) 與權力感知 (Perception of Power)。隱形協調者的存在，即使不直接可見，也能透過其發出的指令（提示）間接影響工作代理的行為，進而影響人類使用者對系統的信任和安全感。這與社會心理學中「權力」和「服從」的概念緊密相關，揭示了AI系統設計中透明度對人類心理的深層影響。
應用場景
當前多代理LLM系統廣泛採用協調者模式以提升效率，例如在客服、內容生成或自動化決策中。本研究揭示了隱式協調可能帶來的安全風險，促使未來在設計這些系統時，需更注重透明度和可解釋性。對心理學實務的啟發是，在設計人機協作系統時，應考慮AI的「組織結構」對人類心理的影響，例如，在心理健康應用中，AI的「領導者」角色應明確且透明，以建立使用者的信任和安全感，避免因隱藏的控制機制而產生焦慮或不適。
隱形指令，無形操控，AI安全警鐘敲響。

Prompt × 心理學 No.03
GraphBit: A Graph-based Agentic Framework for Non-Linear Agent Orchestration
cs.AI
Prompt 技術
顯式圖形化編排 (Explicit Graph-based Orchestration) 取代提示編排 (Prompted Orchestration)。論文指出傳統「提示編排」中，模型本身決定工作流程轉換，常導致幻覺路由和無限循環。GraphBit則透過將工作流程明確定義為有向無環圖（DAG），並由Rust引擎控制路由和狀態轉換，從根本上避免了提示編譯的不可靠性。
心理學概念
認知負荷 (Cognitive Load) 與可預測性 (Predictability)。傳統提示編排的不可預測性（幻覺、循環）會增加開發者和使用者對系統行為的認知負荷，降低信任。GraphBit透過顯式、確定性的圖形化編排，提升了系統的可預測性和可審計性，從而降低了認知負荷，增強了對AI系統的信任感。
應用場景
傳統的LLM代理框架在複雜任務中常因提示編排的不可靠性而受限。GraphBit的出現，為需要高可靠性和可審計性的企業級AI應用提供了解決方案，例如在金融、醫療或法律領域的自動化流程。對心理學實務的啟發是，在設計AI輔助工具時，應優先考慮其行為的確定性和可解釋性。例如，在心理評估或治療輔助系統中，確保AI的決策流程是透明且可追溯的，能有效降低專業人員的焦慮，提升他們對AI工具的接受度和使用意願。
告別提示幻覺，擁抱確定性流程，AI代理更可靠。
