No. 01
OpeFlo: Automated UX Evaluation via Simulated Human Web Interaction with GUI Grounding
cs.HC 2026-04-15
網頁可用性評估傳統上耗時且資源密集，限制了產品開發的迭代速度。本研究提出OpenFlo，一個透過模擬人類網頁行為來自動化使用者體驗評估的AI代理。與依賴DOM解析的傳統工具不同，OpenFlo能基於GUI視覺資訊進行動作與觀察的接地，使其能端到端地與真實網頁互動並維持連貫的使用者旅程追蹤。研究展示了OpenFlo在生成標準化可用性報告方面的有效性，並能識別出傳統方法難以發現的可用性問題。這項創新為小型團隊和敏捷開發流程提供了高效的UX評估解決方案，加速了產品迭代與優化。
如何自動化網頁使用者體驗評估並模擬人類互動行為？
深度點評
  • OpenFlo透過GUI接地而非DOM解析，實現了更接近人類視覺與互動模式的網頁操作，這在技術上是重要的創新，克服了傳統自動化工具的局限性。
  • 相較於現有依賴預設腳本或簡單DOM操作的工具，OpenFlo能生成更真實、連貫的使用者旅程，並能適應網頁動態變化，提升了評估的準確性與廣度。
  • 此研究為人機互動領域的自動化測試與使用者體驗設計提供了強大工具，有望大幅降低UX評估成本，加速產品開發週期，並促進更以使用者為中心的設計流程。
研究貢獻
  • 提出OpenFlo，一個基於GUI接地的自動化UX評估代理。
  • 實現了端到端模擬人類網頁互動的能力，並能追蹤使用者旅程。
  • 展示了在識別可用性問題和生成標準化報告方面的有效性。
研究限制
  • 模擬行為的複雜度可能仍無法完全涵蓋所有人類細微的互動模式。
  • 對於高度動態或非標準化的網頁介面，其穩定性與泛化能力可能仍需進一步驗證。

Automatic Mind Wandering Detection in Educational Settings: A Systematic Review and Multimodal Benchmarking
cs.HC 2026-04-15
在線上教育中，心智遊移（mind wandering）普遍存在，約佔學習時間的30%，嚴重影響學習者的記憶、理解和整體學習成效。本研究旨在透過自動化演算法來偵測心智遊移，以便在適應性學習環境中部署針對性干預措施，從而實現更具回應性和個人化的教育系統。然而，目前缺乏一個連貫的框架來識別線上環境中的心智遊移，阻礙了該領域的進展。本工作提出了一個全面的系統性回顧，並建立了多模態基準測試，旨在統一研究方法並推動自動化心智遊移偵測的發展。
如何在教育環境中自動偵測心智遊移以提升學習成效？
深度點評
  • 該研究不僅進行了系統性回顧，還提出了多模態基準測試，這對於統一不同研究方法、促進領域標準化具有重要意義，有助於克服數據異質性問題。
  • 相較於單一模態或僅限實驗室環境的偵測方法，本研究強調多模態數據整合的潛力，這與真實線上學習環境的複雜性更為契合，提升了實際應用價值。
  • 自動化心智遊移偵測的突破將直接影響適應性學習系統的設計，使教育內容能即時調整以應對學習者的認知狀態，從而顯著提升學習效率與個人化體驗。
研究貢獻
  • 對教育環境中的自動心智遊移偵測進行了全面的系統性回顧。
  • 提出了多模態基準測試框架，以促進研究方法標準化。
  • 強調了心智遊移偵測在適應性學習環境中的關鍵作用。
研究限制
  • 多模態數據的收集與整合在實際部署中仍面臨技術與隱私挑戰。
  • 心智遊移的定義與測量在不同情境下可能存在差異，影響模型的泛化能力。

No. 03
Improving understanding and trust in AI: How users benefit from interval-based counterfactual explanations
cs.HC 2026-04-15
針對黑箱模型的事後解釋（post-hoc explanations）有效性，特別是不同子類型解釋的實驗性使用者研究仍相對缺乏。本研究旨在探討並評估不同類型的反事實解釋（counterfactual explanations），即單點解釋和區間基解釋，如何影響使用者對模型的理解和信任。我們透過一項線上使用者研究，採用受試者內設計，比較了無解釋、特徵重要性解釋、單點反事實解釋和區間基反事實解釋四種條件。研究結果揭示了區間基反事實解釋在提升使用者理解和信任方面的潛在優勢。
不同類型的反事實解釋如何影響使用者對AI的理解與信任？
深度點評
  • 本研究首次系統性地比較了單點與區間基反事實解釋對使用者理解和信任的影響，填補了XAI領域實驗性使用者研究的空白，提供了實證依據。
  • 區間基解釋能提供更全面的決策邊界資訊，這比單點解釋更能反映模型的不確定性與魯棒性，從認知心理學角度看，這有助於使用者建立更精確的心智模型。
  • 研究結果對可解釋AI（XAI）的設計具有直接指導意義，鼓勵開發者採用更具資訊量和認知友善的解釋形式，以促進AI系統在安全關鍵領域的廣泛應用。
研究貢獻
  • 首次實驗性比較了單點與區間基反事實解釋對使用者理解和信任的影響。
  • 發現區間基反事實解釋在提升使用者理解和信任方面具有優勢。
  • 為可解釋AI的設計提供了實證指導。
研究限制
  • 研究僅限於特定任務和模型，結果的泛化性可能受限。
  • 使用者對解釋的偏好和理解可能受個體差異影響，未在研究中深入探討。

如何有效模擬有組織群體的決策行為以理解現實世界動態？
深度點評
  • 本研究將群體行為模擬問題形式化為一個研究平台，並提供了完整的框架、基準和評估標準，這對於推動該領域的標準化研究具有基礎性貢獻。
  • 相較於傳統的個體行為模型或簡化的群體互動模型，本研究更側重於「有組織群體」的決策機制，這在社會心理學和組織行為學上具有更複雜的現實意義。
  • 此研究為社會科學、經濟學和政策制定提供了強大的模擬工具，有助於預測市場反應、評估政策影響，並深入理解複雜社會系統的動態。
研究貢獻
  • 形式化了有組織群體行為模擬問題，並建立了研究平台。
  • 提供了包含任務定義、基準測試和評估標準的框架。
  • 提出了一種新的群體行為模擬演算法並進行了詳細分析。
研究限制
  • 群體行為的複雜性難以完全捕捉，模型可能簡化了某些關鍵的社會互動或決策偏見。

No. 05
Human vs. Machine Deception: Distinguishing AI-Generated and Human-Written Fake News Using Ensemble Learning
cs.CL 2026-04-15
大型語言模型的快速普及催生了一類新型的AI生成假新聞，與傳統人類撰寫的假訊息並存，這引發了關於這兩種欺騙性內容如何區分以及其可靠性如何的重要問題。本研究探討了人類撰寫和AI生成假新聞在語言、結構和情感方面的差異，並評估了機器學習和集成學習方法在區分這些內容類型方面的能力。研究構建了一個文件級的特徵表示，並透過實驗證明了集成學習在辨識AI生成假新聞方面的優勢。
如何有效區分AI生成與人類撰寫的假新聞？
深度點評
  • 本研究首次系統性地比較了AI與人類假新聞在語言、結構和情感維度上的差異，為理解新型假訊息的本質提供了寶貴的洞察，這對社會心理學中的說服與偏見研究有啟發。
  • 相較於僅關注內容真實性的傳統假新聞偵測，本研究將重點放在「來源歸因」，這在資訊戰日益複雜的背景下具有更高的實用價值和挑戰性。
  • 研究成果對於開發更精準的假新聞偵測工具至關重要，有助於維護公共資訊環境的健康，並減輕AI濫用對社會信任和民主進程的潛在威脅。
研究貢獻
  • 系統性比較了AI生成與人類撰寫假新聞的語言、結構和情感差異。
  • 評估了機器學習和集成學習方法在區分這兩類內容上的表現。
  • 構建了文件級特徵表示，並證明了集成學習的有效性。
研究限制
  • AI生成技術不斷演進，模型可能需要持續更新以應對更複雜的生成內容。
  • 研究可能未完全捕捉到所有細微的欺騙策略，特別是針對特定受眾的定制化假新聞。

Self-Calibrating Language Models via Test-Time Discriminative Distillation
cs.CL
核心問題
如何在測試時透過判別式蒸餾來校準大型語言模型的過度自信？
方法亮點
提出一種自校準方法，利用LLM內部已存在的、比其口頭表達更校準的信號（即模型被問及「這個答案是否正確？」時「True」的token機率），並透過測試時判別式蒸餾來改善模型的校準性，無需標註驗證數據。

LLM不再「嘴硬」，自校準讓AI更謙遜、更可靠！

Prompt × 心理學 No.01
Conversational Forecasting Across Large Human Groups Using A Network of Surrogate Agents
cs.CL
Prompt 技術
代理人網路與對話式預測 (Surrogate Agents Network for Conversational Forecasting)。透過部署介入式AI代理，促進大規模分散式人類團隊進行即時對話式協商，並從中匯聚群體智慧進行預測。代理人被設計為引導、總結和促進對話，以克服大規模群體溝通的挑戰。
心理學概念
群體動力學 (Group Dynamics) 與社會影響 (Social Influence)。AI代理的介入旨在優化大規模群體中的溝通流程，減少認知負荷，並促進共識形成，這與社會心理學中關於群體決策、溝通效率和社會影響力的研究高度相關。代理人如何引導對話，會影響個體意見的表達與群體最終的判斷。
應用場景
當前應用於企業決策、市場預測和政策制定等需要大規模群體智慧的場景，能有效提升決策效率與品質。未來潛力在於將此模型推廣至公民參與、危機管理等領域，讓更多元的聲音能被有效整合。這對心理學實務的啟發是，AI可作為中介者，優化人類群體互動，減少偏見，提升集體智慧，但同時需警惕AI引導可能帶來的潛在操縱風險。

Prompt 技術
代理式教學助理框架 (Agentic Teaching Assistant Framework)。該框架包含三個協調模組：檢索增強生成（RAG）的問答系統、自適應測驗生成器和互動式程式碼輔導。透過精心設計的提示，LLM能根據課程材料提供精確、上下文對齊的解釋，並生成針對高階理解的測驗。
心理學概念
認知負荷 (Cognitive Load) 與學習動機 (Learning Motivation)。ACE-TA旨在透過提供即時、個人化的輔導和測驗，降低學習者的認知負荷，並透過互動式學習提升學習動機。其「接地」的問答系統有助於避免AI幻覺，確保學習內容的準確性，這對學習心理學中的知識建構與錯誤修正機制有重要意義。
應用場景
當前已應用於程式設計課程的輔導，提供學生即時的學習支援。未來潛力可擴展至其他學科領域，成為個人化學習的通用工具，甚至在專業培訓中發揮作用。這對心理學實務的啟發是，AI教學助理能有效緩解教師負擔，提供大規模的個性化學習體驗，但需關注AI如何適應不同學習風格，以及過度依賴AI可能對學生自主學習能力造成的影響。
AI助教精準輔導，讓學習不再卡關，程式碼也變有趣！

Prompt × 心理學 No.03
HumorGen: Cognitive Synergy for Humor Generation in Large Language Models via Persona-Based Distillation
cs.CL
Prompt 技術
認知協同框架與人格化蒸餾 (Cognitive Synergy Framework with Persona-Based Distillation)。利用「思維混合」（Mixture-of-Thought, MoT）方法，部署六種認知人格（如荒謬主義者、犬儒主義者）來合成高品質幽默數據，並透過蒸餾將這些幽默生成能力傳授給LLM。這是一種複雜的提示工程，結合了角色扮演與多步驟推理。
心理學概念
幽默認知理論 (Cognitive Theories of Humor) 與人格心理學 (Personality Psychology)。研究直接基於心理學的幽默理論（如不協調理論）來指導AI生成幽默，並透過「人格」的概念來模擬不同風格的幽默感。這深入探討了AI如何理解和生成人類複雜的情感與認知現象，挑戰了AI在創造性思維上的極限。
應用場景
當前應用於提升LLM的幽默生成能力，使其能創造出更具創意和吸引力的文本內容。未來潛力包括在人機互動中增加AI的親和力、用於內容創作（如劇本、廣告文案）以及作為心理治療中的輔助工具（如幽默療法）。這對心理學實務的啟發是，AI能幫助我們更好地理解幽默的認知機制，並提供個性化的幽默內容，但需警惕AI幽默可能帶來的文化敏感性或冒犯性問題。
AI戴上人格面具，幽默感大爆發，讓機器也能講笑話！
