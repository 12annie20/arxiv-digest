AI to Learn 2.0: A Deliverable-Oriented Governance Framework and Maturity Rubric for Opaque AI in Learning-Intensive Domains
cs.CYcs.HC 2026-04-23
生成式AI正以前所未有的速度進入研究、教育和專業領域，但現有的治理框架難以有效評估AI輔助產出的品質。核心問題在於「代理失敗」，即精美的成果可能無法真實反映人類的理解、判斷或知識轉移能力。本研究提出了AI to Learn 2.0框架，旨在為AI輔助工作提供一個以交付物為導向的治理模式。該框架不主張消除AI在學習過程中的作用，而是強調如何確保AI輔助的成果能作為人類學習和能力的可靠證據。透過定義AI輔助工作的成熟度量規，該框架幫助機構評估和提升其AI治理實踐。研究旨在解決AI在學習密集型環境中帶來的評估挑戰，確保AI的整合能真正促進人類發展。這對於教育心理學和AI倫理領域具有重要意義，有助於建立更負責任的AI應用規範。
如何建立AI輔助學習成果的有效治理框架？
深度點評
  • 該框架創新性地將治理重點從AI本身轉向其「交付物」，解決了傳統框架難以評估AI輔助學習成果的困境，提供了一個更實用且以人為本的視角。
  • 相較於現有AI倫理指南多聚焦於技術層面，AI to Learn 2.0更深入地結合了教育心理學的學習評估理論，強調人類理解與判斷的中心地位，填補了AI治理在學習領域的空白。
  • 此框架有望成為教育機構和專業組織在整合生成式AI時的重要參考，促進AI在學習環境中負責任且有益的應用，同時提升對人類核心能力的重視。
研究貢獻
  • 提出以交付物為導向的AI治理框架，解決AI輔助學習成果的評估挑戰。
  • 定義AI輔助工作的成熟度量規，幫助機構評估和提升AI治理實踐。
  • 強調AI輔助成果作為人類理解、判斷和知識轉移能力證據的重要性。
研究限制
  • 框架的實施可能需要大量的培訓和文化轉變，尤其是在評估標準和教師角色方面。
  • 量規的具體應用和效果仍需在不同學習情境中進行廣泛的實證驗證。

如何解決AI自動評分科學解釋中的類別不平衡問題？
深度點評
  • 本研究針對教育領域自動評分系統中常見的類別不平衡問題，系統性地探討了多種資料增強和重採樣策略，為Transformer模型在少數類別上的性能提升提供了實證依據。
  • 相較於僅依賴模型架構改進，本研究從數據層面優化了訓練過程，其方法具有較高的普適性，可應用於其他文本分類任務中的不平衡數據集，尤其是在教育心理學的評估場景。
  • 研究結果對於推動AI在教育評估領域的公平性和準確性具有重要意義，特別是對於捕捉學生高級推理能力的自動化評分，有助於提供更精準的學習回饋。
研究貢獻
  • 系統性評估多種資料增強和重採樣策略在解決文本分類類別不平衡問題上的效果。
  • 改進Transformer模型在自動評分學生科學解釋任務中對少數類別的識別能力。
  • 為NGSS對齊的科學解釋自動評分提供了實用且有效的數據處理方案。
研究限制
  • 資料增強可能引入噪音或不自然的語句，影響模型對真實語料的理解和泛化能力。
  • 研究結果的泛化性可能受限於特定學科和評分標準，需要更多跨領域驗證。

Automated Detection of Dosing Errors in Clinical Trial Narratives: A Multi-Modal Feature Engineering Approach with LightGBM
cs.CLcs.AI 2026-04-23
臨床試驗要求嚴格遵守藥物方案，但劑量錯誤仍然是一個持續存在的挑戰，影響患者安全和試驗完整性。本研究提出了一個自動化系統，用於檢測非結構化臨床試驗敘事中的劑量錯誤。該系統採用梯度提升模型（LightGBM）與全面的多模態特徵工程相結合。我們的方法整合了3,451個特徵，涵蓋了傳統的自然語言處理（TF-IDF、字元n-gram）、密集語義嵌入（all-MiniLM-L6v2）、特定領域的醫學模式以及基於Transformer的模型分數（BiomedBERT、DeBERTa）。這些特徵共同捕捉了文本中的語義、句法和領域特定資訊，以精確識別潛在的劑量錯誤。透過在真實臨床試驗數據集上的評估，該系統展現了高準確性和召回率，顯著提升了劑量錯誤檢測的效率和可靠性。這對於臨床心理學和醫療行為科學領域，特別是藥物依從性和患者安全管理，具有重要的應用價值。
如何自動化檢測臨床試驗敘事中的藥物劑量錯誤？
深度點評
  • 本研究透過結合多模態特徵工程與梯度提升模型，為臨床試驗中的劑量錯誤檢測提供了高效且精確的自動化解決方案，顯著優於單一特徵或模型的方法。
  • 相較於傳統基於規則或簡單NLP的錯誤檢測，本方法整合了深度語義嵌入和領域特定模式，使其能更全面地理解複雜的臨床敘事，提升了檢測的魯棒性與準確性。
  • 該系統的應用潛力巨大，不僅能提升臨床試驗的患者安全性與數據完整性，也能減輕人工審核的負擔，對於臨床心理學在藥物管理和風險評估方面具有直接的實務意義。
研究貢獻
  • 開發基於多模態特徵工程和LightGBM的自動化劑量錯誤檢測系統。
  • 整合多種NLP技術、語義嵌入和領域特定模式，提升檢測準確性。
  • 顯著提高臨床試驗敘事中劑量錯誤檢測的效率和可靠性。
研究限制
  • 模型的性能可能受限於訓練數據的規模和多樣性，對於罕見或新型的劑量錯誤模式可能表現不佳。
  • 系統的解釋性可能不如基於規則的方法直觀，在需要詳細審計的場景中可能面臨挑戰。

Auditing and Controlling AI Agent Actions in Spreadsheets
cs.HCcs.AI 2026-04-23
AI代理的能力進步已超越使用者有效監督其執行的能力。AI代理能夠自主執行複雜的多步驟知識工作，但其執行過程在運行時幾乎無法被使用者存取，通常隱藏在大量的中間推理和輸出中。這導致使用者在收到最終結果時，所有底層決策都已在他們不知情的情況下做出。這種缺乏透明度使得使用者無法檢查代理的假設、識別錯誤或理解其決策邏輯。本研究旨在解決AI代理在試算表環境中執行任務時的透明度不足問題。我們提出了一種新的框架，允許使用者在AI代理執行過程中進行實時審計和控制。透過提供可視化的中間步驟和決策點，使用者可以介入、修正或引導代理的行為，從而增強人機協作的信任和效率。這對於認知心理學中關於信任、控制感和決策支持的研究具有重要啟示。
如何提升使用者對AI代理在試算表操作中的透明度與控制力？
深度點評
  • 本研究針對AI代理在複雜任務中缺乏透明度和使用者控制的問題，提出了一種創新的實時審計與控制框架，有效彌補了現有AI系統在人機協作方面的不足。
  • 相較於傳統的AI事後解釋或黑箱操作，該框架允許使用者在代理執行過程中介入，這不僅提升了使用者對AI的信任感，也符合認知心理學中對控制感和主動參與的需求。
  • 此研究對於提升AI代理在企業和個人生產力工具中的實用性和可接受度具有重要意義，特別是在需要高準確性和可追溯性的知識工作場景中，有助於建立更有效率且以人為中心的AI協作模式。
研究貢獻
  • 提出一個允許使用者實時審計和控制AI代理在試算表操作的框架。
  • 透過可視化中間步驟和決策點，增強AI代理執行的透明度。
  • 提升人機協作的信任和效率，使用戶能介入、修正或引導代理行為。
研究限制
  • 實時介入可能增加使用者的認知負荷，尤其是在代理執行速度較快或步驟複雜時。
  • 框架的設計和實施需要精確平衡自動化效率與使用者控制權，過多的介入可能降低AI的效用。

cs.AIcs.CY 2026-04-23
線上識別支持飲食失調（pro-ED）內容的可靠性面臨兩大普遍問題：現有方法主要依賴文本信號，未能捕捉多媒體內容固有的多模態性質；這些方法難以跟上相關參考、迷因、術語和上下文線索的快速演變。這些限制共同指出了一個空白：缺乏一個由專家註釋的參考標準，能夠支持實時研究和識別不斷演變的pro-ED內容。本研究提出了一個值得深入探討的理念：建立「時代精神感知多模態（ZAM）」數據集，專門用於pro-ED短影片內容。ZAM數據集旨在整合視覺、聽覺和文本等多模態信息，並定期更新以反映網路文化的快速變化。透過專家註釋和持續監測，ZAM數據集將為開發更精確、更及時的pro-ED內容檢測模型提供基礎。這對於臨床心理學、社會心理學和AI倫理領域，特別是青少年心理健康保護，具有極其重要的意義。
如何有效識別線上不斷演變的支持飲食失調多模態內容？

  • 相較於傳統僅依賴文本或靜態數據集的檢測方法，ZAM數據集強調多模態整合和動態更新，更能捕捉網路文化中pro-ED內容的隱蔽性和演變性，顯著提升了檢測的準確性和相關性。
  • 此研究對於臨床心理學和社會心理學在青少年心理健康干預方面具有深遠影響，透過提供更精準的檢測工具，有助於及早發現和干預潛在的飲食失調風險，同時也引發了AI倫理中關於內容監管和言論自由的討論。
研究貢獻
  • 提出「時代精神感知多模態（ZAM）」數據集概念，解決pro-ED內容檢測的多模態和時效性問題。
  • 強調整合視覺、聽覺和文本信息，並定期更新以反映網路文化變化。
  • 為開發更精確、及時的pro-ED內容檢測模型提供專家註釋的參考標準。
研究限制
  • 數據集的建立和維護需要大量的專家資源和持續投入，成本高昂。
  • 內容的敏感性可能引發隱私和倫理問題，需要嚴格的數據管理和使用規範。

LLM × 心理學 No.01
The Tool-Overuse Illusion: Why Does LLM Prefer External Tools over Internal Knowledge?
cs.AIcs.CL
核心問題
大型語言模型為何會過度依賴外部工具而非內部知識？
方法亮點
本研究首先揭示了LLM普遍存在的工具過度使用現象，即在推理過程中不必要地使用外部工具。接著，透過分析LLM在不同內部知識可用性區域的工具使用行為，實驗性地闡明了其潛在機制。研究識別出一個「知識認知錯覺」，即模型錯誤判斷其內部知識的可用性或適用性，導致即使內部知識足以解決問題時，仍傾向於尋求外部工具的協助。實驗設計精巧，透過控制內部知識的豐富程度和外部工具的易用性，觀察LLM的決策模式。
心理學意涵
這項研究揭示了LLM在認知層面可能存在的「知識認知錯覺」，這與人類在決策時的認知偏誤有異曲同工之妙。它挑戰了我們對LLM理性行為的假設，暗示模型可能存在類似人類的「確認偏誤」或「外部化傾向」。這對認知心理學理解人工智慧的「心智」運作提供了新的視角，也對AI設計者提出了警示，需在模型架構和訓練中考慮如何平衡內部知識與外部工具的使用，以避免效率低下或錯誤決策。此外，這也引發了對AI信任度的思考，當AI過度依賴外部工具時，其決策的自主性和可靠性可能受到質疑。
LLM的「工具過度使用錯覺」揭示了其深層認知偏誤，值得我們重新審視AI的「理性」邊界。

LLM × 心理學 No.02
From Actions to Understanding: Conformal Interpretability of Temporal Concepts in LLM Agents
cs.AIcs.CL
核心問題
如何解釋大型語言模型代理在多步驟推理和決策中的時間概念演變？
方法亮點
本研究提出了一個框架，透過逐步的共形視角來解釋LLM代理中時間概念的演變。該框架結合了逐步獎勵分解和共形預測，以量化模型在每個時間步驟中對特定概念的信心。透過分析代理在互動環境中的序列行為，研究能夠追蹤關鍵概念（如目標、狀態、動作）的內部表徵如何隨時間變化。這種方法允許研究者在不修改模型內部結構的情況下，對其決策過程進行可信賴的解釋，從而揭示其不透明的內部機制。
心理學意涵
這項研究對於理解LLM代理的「心智」如何處理時間序列信息和進行決策具有重要意義。它提供了一種量化和解釋AI代理內部認知過程的方法，有助於彌合AI黑箱與人類理解之間的鴻溝。在認知心理學中，理解時間推理和概念演變是核心議題，本研究為此提供了計算模型層面的洞察。未來，這可以應用於設計更具可解釋性和可信賴性的AI代理，尤其是在需要長期規劃和複雜決策的場景中，例如教育輔導、臨床診斷或個人助理。同時，這也為研究人類與AI代理的協作模式提供了基礎，理解AI如何「理解」時間，將有助於優化人機互動。

Semantic Prompting: Agentic Incremental Narrative Refinement through Spatial Semantic Interaction
cs.CLcs.HC
Prompt 技術
本研究提出「語義提示（Semantic Prompting）」技術，透過空間語義互動實現代理式增量敘事精煉。使用者可以透過互動式空間佈局來組織資訊和發現，並利用LLM自動生成敘事。該技術的核心是解決互動-修訂錯位、人-LLM意圖錯位和缺乏細粒度客製化問題，允許使用者在空間佈局上進行增量修改，並透過語義提示引導LLM進行精確的敘事更新，而非完全重新生成。
心理學概念

應用場景

語義提示讓LLM敘事生成更貼近人類思維，有效降低認知負荷，提升創作控制感。

Prompt × 心理學 No.02
Saying More Than They Know: A Framework for Quantifying Epistemic-Rhetorical Miscalibration in Large Language Models
cs.CLcs.AI
Prompt 技術
本研究並非直接使用特定的提示工程技術，而是透過設計一個「三元認知-修辭標記（ERM）」分類法，來量化LLM的「認知-修辭失準」。這可以被視為一種評估LLM輸出品質的「提示設計」策略，即透過特定的提示來引導LLM生成不同修辭強度的文本，然後分析其與認知基礎的匹配程度。研究透過複合指標（如形式-意義分歧FMD、真實-表現認知比GPR、修辭裝置分佈熵RDDE）來衡量這種失準。
心理學概念
涉及「認知偏誤」、「說服」和「信任」。LLM的「認知-修辭失準」反映了模型在表達自信程度時與其實際知識基礎的不匹配，這類似於人類的「過度自信偏誤」。這種失準可能影響使用者對LLM輸出的「信任」，因為過於自信的表達可能具有更強的「說服力」，即使內容不準確。這對社會心理學中關於說服和信任的研究提供了AI層面的新視角。
應用場景
當前應用於評估LLM在生成論證性文本時的可靠性和誠實性。未來潛力在於開發能自動檢測和校準LLM修辭強度的工具，使其輸出更符合實際的認知基礎。對心理學實務的啟發是，這有助於心理學家理解AI在溝通中可能產生的誤導性，並設計更有效的提示策略來引導AI生成更客觀、更負責任的資訊，尤其是在心理健康資訊提供或輔助決策的場景中，避免因AI的「過度自信」而產生誤判。
LLM的「言過其實」是認知偏誤的AI版，量化它有助於建立更誠實的AI溝通。

Prompt × 心理學 No.03
Cognis: Context-Aware Memory for Conversational AI Agents
cs.CLcs.AI
Prompt 技術
本研究提出Lyzr Cognis，一個用於對話式AI代理的統一記憶架構，透過多階段檢索管道解決LLM缺乏持久記憶的問題。其方法亮點在於結合了雙儲存後端（OpenSearch BM25關鍵字匹配與Matryoshka向量相似性搜索），並透過倒數排名融合（Reciprocal Rank Fusion）進行整合。其「情境感知攝取管道」在提取新記憶之前檢索現有記憶，實現智能版本追蹤和記憶更新，這是一種高級的RAG（Retrieval-Augmented Generation）策略，用於管理和優化LLM的上下文。
心理學概念
涉及「記憶」、「情境感知」和「個性化」。Lyzr Cognis的設計靈感來自人類的記憶系統，旨在賦予AI代理類似人類的長期記憶能力，使其能夠在對話中保持「情境感知」，並實現「個性化」互動。這與認知心理學中關於工作記憶、長期記憶和情境記憶的研究緊密相關，探討了如何為AI構建更接近人類的記憶模型。
應用場景
當前應用於提升對話式AI代理的持久記憶和個性化對話能力，例如智能客服、個人助理和教育輔導機器人。未來潛力在於創造更具「人情味」和「理解力」的AI伴侶，能夠記住使用者的偏好、歷史互動和個人資訊。對心理學實務的啟發是，這種具備持久記憶的AI可以作為心理健康支持工具，提供更連貫、更個性化的輔導和陪伴，例如記住患者的治療進度、情緒變化和個人故事，從而提供更具同理心的互動，但同時也需關注隱私和數據安全問題。
賦予AI持久記憶，讓對話更具人情味，但隱私與安全仍是心理學的考驗。
