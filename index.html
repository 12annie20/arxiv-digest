No. 01
CreativityBench: Evaluating Agent Creative Reasoning via Affordance-Based Tool Repurposing
cs.AI 2026-05-07
大型語言模型（LLMs）在推理和環境互動任務上展現出強大性能，但其創造性問題解決能力仍未被充分探索。本研究透過創造性工具使用的視角來探討此能力，其中模型需透過推理物體的「可供性」（affordances）和屬性來重新利用現有物品，而非僅依賴其傳統用途。作為首要步驟，研究團隊引入了CreativityBench，一個用於評估LLM基於可供性的創造力的基準測試。為此，他們建立了一個大規模的可供性知識庫，並設計了多樣化的任務情境。實驗結果顯示，儘管LLMs在某些方面有所進步，但在複雜的創造性工具重用任務中仍面臨挑戰。這項工作為理解和提升AI的創造性推理能力奠定了基礎，並指出了未來研究的方向。
如何評估AI代理人基於可供性的創造性推理能力？
深度點評
  • 本研究透過引入「可供性」（affordance）概念來評估AI的創造力，這是一個新穎且具心理學基礎的方法，超越了傳統基於文本生成的創造力評估。然而，其知識庫的構建和任務設計可能仍難以完全捕捉人類創造性思維的複雜性和情境依賴性。
  • 相較於現有僅依賴語言模型生成或簡單組合的創造力評估，CreativityBench更強調AI對物理世界物件功能潛力的理解與重構，這代表了AI創造性研究的一個重要進展。但如何確保基準測試的普適性和避免過度擬合特定任務仍是挑戰。
  • 這項研究對AI領域的潛在影響巨大，它不僅為評估AI的創造性提供了一個具體框架，也可能激發更多關於如何訓練AI進行非傳統、情境化推理的研究，進而推動通用人工智慧的發展。
研究貢獻
  • 引入CreativityBench基準測試，用於評估LLM基於可供性的創造性工具重用能力
  • 建立大規模可供性知識庫，支持創造性推理任務的設計
  • 透過實驗揭示LLM在創造性問題解決方面的潛力與局限
研究限制
  • 可供性知識庫的完整性與泛化能力可能仍有不足
  • 當前任務設計可能未能完全捕捉人類創造性思維的全部複雜性

No. 02
Stop Automating Peer Review Without Rigorous Evaluation
cs.AIcs.CY 2026-05-07
大型語言模型（LLMs）為解決學術同儕審查危機提供了誘人的解決方案，但本立場文件強烈主張，目前的AI系統不應被用於生成論文審查意見。研究團隊透過對人類與AI生成的ICLR 2026審查意見進行實證比較，並評估自動論文重寫對不同AI審查者的影響，來支持這一觀點。他們發現了兩個關鍵問題：首先，AI審查者表現出「蜂巢思維效應」（hivemind effect），即在論文內部和跨論文之間存在過度一致性，這降低了觀點的多樣性。其次，AI審查分數容易受到論文重寫的操縱，導致其可靠性受損。這些發現強調了在將AI應用於學術審查等高風險領域時，必須進行嚴格評估和謹慎考量。
AI系統是否適合自動化學術同儕審查？
深度點評
  • 本研究對AI自動化同儕審查提出了嚴肅的倫理與實用性質疑，特別指出了AI審查意見的「蜂巢思維效應」，這揭示了當前AI模型在生成多樣化、批判性觀點方面的根本性局限。這對AI在決策支持系統中的應用具有普遍的警示意義。
  • 相較於僅探討AI審查效率或語法流暢度的研究，本文更深入地觸及了AI審查的「品質」與「多樣性」問題，並透過實證比較揭示了AI審查分數易受操縱的脆弱性。這為學術界在考慮AI輔助審查時提供了重要的決策依據。
  • 這項研究對學術出版和AI倫理領域具有深遠影響。它不僅呼籲停止未經嚴格評估的AI審查自動化，也促使人們重新思考AI在需要高度判斷力、多樣化視角和抗操縱性的任務中的角色與界限。
研究貢獻
  • 實證比較人類與AI生成的學術審查意見
  • 揭示AI審查者存在的「蜂巢思維效應」與觀點多樣性不足問題
  • 證明AI審查分數易受論文重寫操縱，影響其可靠性
研究限制
  • 研究僅限於ICLR 2026的審查，結果的泛化性可能受限
  • 對「蜂巢思維效應」的深層機制探討仍有待加強

No. 03
Toward Human-AI Complementarity Across Diverse Tasks
cs.AIcs.HC 2026-05-07
人機互補性，即結合人類與AI判斷能超越單獨一方的表現，為先進AI系統的穩健監督提供了一條有前景的途徑。然而，人機互補性是否能在實際任務中實現仍是一個開放性問題。本研究透過兩種方法來探討這一點：混合式方法（hybridization）和兩種AI輔助方法（top-2輔助和子任務委託），並在一個包含1,886個樣本的多領域數據集上進行評估，這些樣本涵蓋了知識、事實性、長上下文推理和欺騙檢測。研究發現，在多數任務中，人機互補性確實能夠提升整體性能，尤其是在AI表現不佳或人類判斷力較強的領域。這項工作為設計更有效的人機協作系統提供了實證基礎，並強調了AI在不同任務中扮演輔助角色的潛力。
如何在多樣化任務中實現人機互補性以提升性能？
深度點評
  • 本研究系統性地探討了人機互補性在多樣化任務中的實現，特別是提出了混合式和AI輔助（top-2、子任務委託）兩種具體方法，這為人機協作的設計提供了實用的策略。然而，如何動態地判斷何時採用何種輔助模式，以及如何量化人類與AI各自的「優勢區」，仍是未來研究的重點。
  • 相較於過往多數集中於單一任務或領域的人機協作研究，本研究涵蓋了知識、事實性、長上下文推理和欺騙檢測等多個複雜任務，展現了其方法的普適性。這為構建更通用、適應性強的人機協作框架奠定了基礎，但數據集的多樣性與真實世界複雜度的匹配仍需進一步驗證。
  • 這項研究對人機互動和AI系統設計具有重要影響。它不僅提供了實現人機互補性的實證證據和具體方法，也為未來開發能夠智能判斷何時介入、如何輔助人類的AI系統指明了方向，有助於提升AI在複雜決策場景中的應用價值。
研究貢獻
  • 提出並評估了混合式與AI輔助兩種人機互補性實現方法
  • 在涵蓋知識、事實性、推理和欺騙檢測的多領域數據集上進行了廣泛驗證
  • 提供實證證據，證明人機互補性在多數任務中能提升性能
研究限制
  • 研究中AI輔助方法的選擇和參數設定可能影響互補性效果
  • 數據集雖然多樣，但仍可能無法完全代表所有實際應用場景的複雜性

Characterizing Students' LLM Usage Behaviors and Their Association with Learning in Critical Thinking Tasks
cs.AIcs.HC 2026-05-07
大型語言模型（LLMs）日益融入學生的學習實踐中，然而，關於學生如何使用LLMs以及這種使用如何影響學習的知識，大多來自於問題解決領域或受限的實驗環境。本研究分析了在兩次以研究為導向的課程中收集到的LLM使用數據，學生在課程中學習閱讀、推理和批判學術論文。在沒有限制LLM使用方式的情況下，學生被要求報告他們的LLM使用習慣。研究結果揭示了學生在批判性思維任務中多樣化的LLM使用模式，並探討了這些行為與學習成果之間的關聯。這項工作為理解LLM在真實學習環境中的影響提供了寶貴的見解，並為教育者和AI設計者提供了指導。
學生在批判性思維任務中如何使用LLM，及其對學習的影響？
深度點評
  • 本研究在真實、非受限的學習環境中觀察學生LLM使用行為，這比傳統實驗室研究更具生態效度，能更真實地反映LLM對學習的影響。然而，學生自我報告的使用習慣可能存在偏差，且學習成果的衡量方式也可能影響結果的解釋。
  • 相較於多數關注LLM在編程或寫作等特定技能輔助的研究，本研究聚焦於LLM對「批判性思維」這一高階認知能力的影響，這是一個更具挑戰性且重要的領域。但如何區分LLM輔助下的「思考」與「抄襲」仍是教育界的一大難題。
  • 這項研究對教育心理學和學習科技領域具有重要意義。它不僅為教育者提供了關於如何指導學生有效利用LLM的實證依據，也為設計能促進而非阻礙批判性思維發展的AI學習工具提供了啟示，有助於應對AI時代的教育挑戰。
研究貢獻
  • 在真實課堂環境中收集並分析學生LLM使用行為數據
  • 揭示學生在批判性思維任務中多樣化的LLM使用模式
  • 探討LLM使用行為與學習成果之間的關聯
研究限制
  • 學生LLM使用行為主要依賴自我報告，可能存在回憶偏差
  • 研究結果的因果關係難以完全確立，可能存在其他混淆變量

Cognitive Alignment Drives Attention: Modeling and Supporting Socially Shared Regulation in Pair Programming
cs.CYcs.HC 2026-05-07
本論文以社會共享學習調節（SSRL）為基礎，探討了聯合心智努力（JME）和聯合視覺注意力（JVA）如何作為結對編程中共享調節的過程級指標，以及AI驅動的自適應回饋如何強化這些過程。研究團隊進行了三項眼動追蹤研究，涉及182對參與協作調試任務的學生。研究一檢視了自然協作，結果顯示高績效的配對展現出顯著更高的JME和JVA，以及更頻繁的富有成效的高JME。研究二和研究三則進一步探討了AI輔助回饋對JME和JVA的影響。這些發現不僅加深了對協作學習中認知對齊機制的理解，也為設計能有效支持團隊學習的AI系統提供了實證依據。
認知對齊如何驅動注意力並支持結對編程中的共享調節？
深度點評
  • 本研究創新性地將聯合心智努力（JME）和聯合視覺注意力（JVA）作為社會共享學習調節的過程指標，並透過眼動追蹤進行量化，這為理解協作學習的微觀動態提供了精確的數據。然而，JME的測量仍可能存在主觀性，且AI回饋的設計複雜性也限制了其在不同情境下的泛化能力。
  • 相較於傳統僅依賴問卷或訪談的協作學習研究，本研究結合了眼動追蹤等生理測量技術，提供了更客觀、即時的數據來分析認知對齊與注意力。這為教育心理學和人機互動領域的交叉研究樹立了典範，但大規模部署和倫理考量仍是挑戰。
  • 這項研究對教育科技和AI輔助學習系統的發展具有重要影響。它不僅揭示了認知對齊在協作學習中的關鍵作用，也為設計能夠即時監測並提供自適應回饋的AI導師系統提供了理論和實證基礎，有助於提升團隊學習效率和品質。
研究貢獻
  • 將聯合心智努力（JME）和聯合視覺注意力（JVA）作為社會共享學習調節的過程指標
  • 透過三項眼動追蹤研究，實證JME和JVA與高績效協作的關聯
  • 探討AI驅動的自適應回饋如何強化協作學習中的共享調節
研究限制
  • JME的測量可能仍存在一定程度的主觀性
  • AI回饋的設計和實施可能需要針對不同任務和學習者進行調整

LLM × 心理學 No.01
ADAPTS: Agentic Decomposition for Automated Protocol-agnostic Tracking of Symptoms
cs.AIcs.CL
核心問題
如何利用LLM自動且不依賴特定協議地追蹤臨床症狀嚴重程度？
方法亮點
ADAPTS框架採用混合代理人（mixture-of-agents）LLM架構，將長篇臨床訪談分解為症狀特定的推理任務。這種方法能產生可審計的理由，同時保留時間和說話者對齊。模型在兩個獨立的數據集上進行泛化評估，以確保其在不同臨床情境下的有效性。
心理學意涵
這項研究對臨床心理學和精神醫學領域具有重大意義，它提供了一種自動化、標準化的工具來評估患者的抑鬱和焦慮嚴重程度，有助於減輕臨床醫生的工作負擔。透過分解任務和提供可審計的理由，ADAPTS提升了AI在敏感臨床應用中的透明度和可信度，這對於建立患者和醫生的信任至關重要。然而，AI在情感計算和症狀評估中的倫理問題，如數據隱私、偏見和誤診風險，仍需持續關注和嚴格監管。未來可探索將此技術應用於遠程醫療和早期篩查，以擴大心理健康服務的可及性。

AICoFe: Implementation and Deployment of an AI-Based Collaborative Feedback System for Higher Education
cs.AIcs.HC
核心問題
如何透過AI輔助提升高等教育中同儕回饋的品質與效率？
方法亮點
AICoFe系統採用以人為中心的AI方法，實施模組化架構，協調多個LLM（如GPT-4.1-mini, Gemini 2.5 Flash, Llama 3.1）的管線。該系統旨在綜合量化評分標準數據和定性觀察，以生成高質量的同儕回饋，從而彌補學生生成回饋品質不一的局限性。
心理學意涵
