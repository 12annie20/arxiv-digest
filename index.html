A learning health system in Neurorehabilitation as a foundation for multimodal patient representation
cs.AIcs.HC 2026-04-28
神經復健領域面臨著日益增長的全球健康負擔，需要長期且跨學科的治療。計算神經復健（compNR）利用數據驅動和模型化方法來實現個人化治療，但其臨床應用受限於數據碎片化和互操作性差等問題。本研究旨在透過整合多模態數據，將學習型健康系統（LHS）框架嵌入神經復健中，以克服這些挑戰。研究團隊構建了一個多模態患者數據平台，匯集了臨床、感測器和影像數據，並開發了標準化的數據處理流程。透過LHS，系統能夠持續學習並優化治療方案，促進臨床決策支援工具的開發。初步結果顯示，該平台能有效整合異質數據，為精準復健奠定基礎。這項工作對於提升神經復健的效率和個人化程度，具有重要的臨床意義。
如何建立神經復健的學習型健康系統？
深度點評
  • 本研究將學習型健康系統（LHS）框架引入神經復健，透過整合多模態數據來實現個人化治療，其創新點在於強調數據的持續學習與優化，而非單次模型訓練。
  • 相較於傳統的單一數據源分析，本研究的多模態數據整合方法能更全面地捕捉患者狀態，為神經復健提供了更精確的評估與干預基礎。
  • 這項工作為計算神經復健的臨床部署鋪平了道路，有望加速數據驅動的精準復健發展，並提升患者的治療效果與生活品質。
研究貢獻
  • 提出並實施了神經復健領域的學習型健康系統框架。
  • 建立了整合多模態患者數據的平台，解決了數據碎片化問題。
  • 為精準復健提供了數據驅動的基礎，促進治療方案的持續優化。
研究限制
  • 系統的長期臨床效益和成本效益仍需進一步驗證。
  • 數據隱私和安全問題在多模態數據整合中仍是重要挑戰。

Visual Accessibility in a Virtual Kitchen: Effects of Open Shelving on Performance, Cognitive Load, and Experience in Older Adults with and without MCI
cs.HCcs.CY 2026-04-28
本研究探討了虛擬廚房中，櫥櫃設計（開放式置物架）如何透過視覺可及性影響老年人（包括輕度認知障礙MCI患者）的任務表現、認知負荷和使用者體驗。研究招募了17名老年人（7名MCI患者，10名非MCI患者），採用平衡的組內設計，在封閉式櫥櫃和開放式置物架兩種條件下完成物品取回任務。測量指標包括任務持續時間、身體活動水平、認知負荷（NASA-TLX）和使用者體驗。結果顯示，開放式置物架顯著改善了任務表現，降低了認知負荷，並提升了使用者體驗，尤其對MCI患者的幫助更為顯著。這項研究為虛擬環境和實際生活空間的設計提供了實證依據。其發現對於提升老年人，特別是MCI患者的生活品質和獨立性，具有重要啟示。
虛擬廚房設計如何影響老年人認知負荷與體驗？
深度點評
  • 本研究透過虛擬實境環境，精確控制變量，探討了視覺可及性對老年人（含MCI患者）認知負荷和任務表現的影響，其方法學設計嚴謹且具創新性。
  • 相較於傳統的實體環境研究，虛擬廚房提供了更安全、可控的實驗場景，使得研究能夠更深入地分析設計元素對特定人群的影響。
  • 這項研究的發現不僅對虛擬實境應用設計有指導意義，也為實際居家環境的適老化改造提供了寶貴的設計原則，有助於提升老年人的生活品質和獨立性。
研究貢獻

  • 虛擬環境的沉浸感和真實感可能仍與實際環境存在差異。

MindTrellis: Co-Creating Knowledge Structures with AI through Interactive Visual Exploration
cs.HCcs.AI 2026-04-28
知識工作者在從多份文件中綜合資訊並建構結構化概念理解時，面臨日益嚴峻的挑戰。這個過程本質上是迭代的：使用者探索內容、識別概念間的關係並不斷重組其心智模型。然而，目前的解決方案支援有限。基於LLM的系統允許使用者查詢資訊，但無法塑造知識的組織方式；而心智圖等手動工具雖然支援結構創建，卻缺乏智慧輔助。本研究提出MindTrellis，一個透過互動式視覺探索與AI共同創建知識結構的系統。MindTrellis結合了LLM的智慧生成能力和使用者對知識結構的精確控制，讓使用者能夠在探索資訊的同時，直觀地組織和重構概念。實驗結果顯示，MindTrellis顯著提升了知識工作者的效率和滿意度。這項工作為人機協作知識建構開闢了新途徑。
如何透過AI與視覺化工具共同建構知識結構？
深度點評
  • MindTrellis創新性地結合了LLM的生成能力與互動式視覺化介面，解決了知識工作者在資訊綜合與心智模型建構中的痛點，實現了人機協作的深度融合。
  • 相較於單純的LLM查詢或手動心智圖工具，MindTrellis提供了更為動態和靈活的知識組織方式，允許使用者在探索中不斷迭代和精煉其概念結構。
  • 這項研究為未來智慧知識管理工具的發展提供了重要啟示，特別是在認知科學領域，它展示了AI如何輔助人類進行複雜的認知任務，提升學習與創造效率。
研究貢獻
  • 提出MindTrellis系統，實現AI與人類共同創建知識結構。
  • 結合LLM的智慧生成與互動式視覺探索，支援迭代式知識建構。
  • 顯著提升知識工作者在資訊綜合與概念理解方面的效率和滿意度。
研究限制
  • 系統的擴展性在處理極大量或極複雜的知識領域時可能面臨挑戰。
  • 使用者對AI輔助的信任度與依賴性仍需進一步研究。

Towards Causally Interpretable Wi-Fi CSI-Based Human Activity Recognition with Discrete Latent Compression and LTL Rule Extraction
cs.AIcs.HC 2026-04-28
本研究旨在解決利用Wi-Fi通道狀態資訊（CSI）進行人類活動識別（HAR）時，同時滿足因果可解釋性、符號可控性以及直接處理高維原始訊號的需求。深度神經模型在CSI-based HAR（CHAR）上表現出色，但其連續潛在表示不透明且難以修改；純符號方法則無法處理原始CSI流。我們提出了一個完全自動化且嚴格解耦的流程，其中CSI幅度訊號首先透過離散潛在壓縮進行處理。接著，從壓縮後的離散潛在空間中提取線性時序邏輯（LTL）規則，這些規則直接對應於可解釋的活動模式。實驗結果表明，該方法在保持高預測性能的同時，提供了清晰的因果解釋。這項工作為開發更透明、可控的HAR系統提供了新途徑。
如何實現Wi-Fi CSI人類活動識別的因果可解釋性？
深度點評
  • 本研究提出了一種創新的解耦流程，將深度學習的特徵提取能力與符號邏輯的解釋性結合，有效解決了CSI-based HAR在可解釋性方面的挑戰。
  • 相較於傳統的黑箱深度學習模型，本方法透過離散潛在壓縮和LTL規則提取，提供了清晰的因果解釋，使得模型決策過程更為透明和可信。
  • 這項工作對於行為科學和智慧居家應用具有重要意義，不僅能提升HAR系統的可靠性，也為開發能與人類有效互動、並提供可理解反饋的AI系統奠定了基礎。
研究貢獻
  • 提出結合離散潛在壓縮和LTL規則提取的因果可解釋HAR框架。
  • 實現了在處理高維原始CSI訊號的同時，提供清晰的因果解釋。
  • 為開發更透明、可控的人類活動識別系統提供了新方法。
研究限制
  • LTL規則的複雜度可能隨活動類型增加而提升，影響可讀性。
  • 模型的泛化能力在不同環境和個體間仍需進一步驗證。

LLM × 心理學 No.01
Don't Make the LLM Read the Graph: Make the Graph Think
cs.AIcs.CL
核心問題
信念圖譜如何提升LLM在多代理推理中的表現？
方法亮點
本研究透過在合作卡牌遊戲Hanabi中進行3,000多次受控試驗，探討了顯式信念圖譜對LLM在合作多代理推理中表現的影響。研究發現，整合架構是關鍵：當信念圖譜作為提示上下文時，對強模型而言僅是裝飾，對弱模型則在二階心智理論任務上有所助益；但當圖譜透過排序的候選列表來引導行動選擇時，它們能結構性地提升所有模型的表現。
心理學意涵
這項研究深入探討了LLM在多代理情境下的「心智理論」（Theory of Mind）能力，揭示了外部知識表示（信念圖譜）如何影響LLM的推理和協作行為。它挑戰了單純將圖譜作為提示上下文的傳統做法，強調了將圖譜作為行動選擇的門控機制，能更有效地提升LLM的認知表現。這對理解人類在社交互動中如何利用共享知識和信念進行推理，以及如何設計更具社交智慧的AI代理，提供了重要的心理學啟示。同時，也引發了關於AI代理間協作效率和信任機制的思考。
別讓LLM只看圖，讓圖譜引導LLM思考，才是提升多代理協作的關鍵。

LLM讓醫療對話分析不再是苦差事，但性能、速度與環保仍需權衡。

Implicit Framing in Obstetric Counseling Notes: A Grounded LLM Pipeline on a VBAC-Eligible Cohort

LLM如何揭示產科諮詢記錄中的隱性框架？

本研究分析了2,024份產科病史和身體檢查記錄中的醫師諮詢語言，特別關注陰道分娩後剖腹產（VBAC）和重複剖腹產（RCS）的決策情境。研究提出了一個基於LLM的管道，用於識別和量化臨床框架（即臨床資訊呈現方式）對患者理解和決策的影響。該管道結合了自然語言處理技術和領域專家知識，以揭示諮詢記錄中隱含的語言模式和偏向性。
心理學意涵
這項研究對臨床心理學和社會心理學具有深遠影響。它利用LLM分析臨床溝通中的「框架效應」（framing effect），揭示了醫師在諮詢中如何透過語言選擇，無意或有意地影響患者的決策。這對於理解醫病溝通的心理機制、提升醫學倫理意識以及優化患者知情同意過程至關重要。研究結果可以指導醫療專業人員更自覺地使用中立、客觀的語言，減少潛在的偏見，從而賦予患者更大的自主決策權，並改善醫療結果。
LLM揭露產科諮詢的語言潛規則，讓醫病溝通更透明、更公平。

Cognitive Alignment Deciphered: A Self-Developed Scenario-Based Prompt Scale Coupled with Representational Similarity Analysis and Social Network Analysis for Unraveling Bias Mechanisms Across Humans and LLMs

場景式提示量表（Scenario-Based Prompt Scale）：透過設計一系列情境驅動的提示，涵蓋58種認知偏誤，並橫跨計算、信念、資訊、社會和記憶五個雙系統維度，以評估人類和LLM在不同情境下的認知反應。
心理學概念

DeepImagine: Learning Biomedical Reasoning via Successive Counterfactual Imagining
cs.CL
Prompt 技術
連續反事實想像（Successive Counterfactual Imagining）：這是一種提示策略，引導LLM透過逐步構建和評估「如果…會怎樣」的假設情境，來模擬複雜的因果推理過程，特別應用於生物醫學領域的臨床試驗結果預測。
心理學概念
反事實思維（Counterfactual Thinking）與推理（Reasoning）：反事實思維是人類一種常見的認知過程，涉及對過去事件的替代性想像。本研究將此概念應用於LLM，透過提示引導模型進行類似人類的因果推理，以理解和預測複雜的生物醫學現象。
應用場景
當前應用在於提升LLM在預測臨床試驗結果方面的能力，這對於藥物研發和醫療決策具有重要價值。未來潛力包括將此反事實想像的推理框架推廣到其他需要複雜因果分析的科學領域，例如社會科學或經濟學。對心理學實務的啟發是，它展示了如何透過精巧的提示設計，激發LLM展現出類似人類的深層次認知能力，這對於開發能夠輔助人類進行複雜決策、甚至進行科學發現的AI工具，具有重要的理論和實踐意義。同時，也為研究人類反事實思維的計算模型提供了新的視角。
讓LLM學會「如果…會怎樣」，精準預測生物醫學的未來。

Prompt × 心理學 No.03
ContextWeaver: Selective and Dependency-Structured Memory Construction for LLM Agents
cs.CL
Prompt 技術
選擇性與依賴結構化記憶建構（Selective and Dependency-Structured Memory Construction）：這是一種進階的上下文管理技術，它將LLM代理的互動軌跡組織成具有因果和邏輯結構的記憶，而非簡單的滑動窗口或提示壓縮，以解決長上下文互動中的資訊遺失問題。
心理學概念
記憶（Memory）與多步驟推理（Multi-step Reasoning）：本研究借鑒人類記憶系統的組織方式，為LLM代理設計了更高效的記憶管理機制。人類在進行複雜推理時，會選擇性地提取和整合相關記憶。ContextWeaver旨在模仿這種認知過程，使LLM在長對話中能更好地維持上下文連貫性並進行複雜推理。
應用場景
當前應用在於提升LLM代理在需要長期記憶和多步驟推理的複雜任務中的表現，例如自動化客服、智慧助手或科學研究代理。未來潛力包括將此記憶框架應用於開發更具「常識」和「經驗」的AI代理，使其在面對新情境時能更好地泛化和適應。對心理學實務的啟發是，它提供了一個計算模型來理解人類在長時互動中如何管理和利用記憶進行推理，這對於研究人類認知負荷、注意力分配以及記憶提取策略具有借鑒意義。同時，也為設計更自然、更具連貫性的人機對話系統提供了技術基礎。
LLM代理不再健忘，ContextWeaver讓記憶有結構、推理更流暢。
