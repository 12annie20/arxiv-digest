No. 01
Toward Accessible Psychotherapy Training Using AI-Driven Interactive Patient Avatars
cs.AIcs.HC 2026-06-18
心理治療師的訓練需要大量實踐與回饋，但倫理、物流與資源限制使得安全且標準化的訓練機會稀少。本研究提出一個系統，旨在透過與具身虛擬病患的口語對話，支援以「接受與承諾療法」(ACT)為導向的心理治療訓練。該系統利用大型語言模型，根據真實治療會話衍生的病患檔案與可配置的臨床情境來模擬病患行為。這使得治療師能在受控環境中練習，並獲得即時回饋。研究結果顯示，此系統能有效提升治療師的實踐能力與信心。這項創新為心理治療教育提供了可擴展且高效的解決方案。
AI如何提供可近且標準化的心理治療訓練？
深度點評
  • 該系統利用LLM模擬病患行為，並結合真實治療會話數據建立病患檔案，這在模擬真實性上是一大進步。然而，LLM在處理複雜情緒與非語言線索的細微差別上仍有侷限，可能影響訓練的全面性。
  • 相較於傳統的角色扮演或有限的虛擬實境訓練，本研究的系統提供了更具互動性、可配置性與標準化的訓練環境，顯著提升了訓練的可擴展性與效率。
  • 此研究為心理治療教育開闢了新途徑，有望降低訓練成本、提高訓練品質，並最終擴大心理健康服務的普及性。
研究貢獻
  • 開發了基於LLM的具身虛擬病患系統
  • 實現了ACT導向的心理治療訓練模擬
  • 提供了可配置的臨床情境與病患行為模擬
研究限制
  • LLM在模擬複雜人類情感與非語言溝通上的挑戰
  • 系統對真實治療情境的覆蓋範圍有限

No. 02
The Bias Paradox: How AI Personas Can Overcome Human Limitations in UX Research
cs.HCcs.AI 2026-06-18
本立場論文探討了UX研究實踐中遇到的「偏見悖論」：在特定情境下，真實人類參與者因情境誘導偏見，反而不如AI人格提供真實洞察。作者分享了使用OpenAI的客製化GPT建構器開發研究型AI人格的經驗，並在與高淨值銀行客戶的設計思考工作坊中進行了驗證。工作坊的豪華酒店環境、在場的投資組合經理以及款待氛圍，都引入了偏見，損害了回饋的真實性。研究提出AI人格能提供更客觀、無偏見的數據，從而克服人類參與者的局限性。這項發現挑戰了傳統UX研究方法，並為未來研究提供了新方向。
AI人格如何克服人類偏見以提升UX研究的真實性？
深度點評
  • 論文提出AI人格作為UX研究工具，其創新點在於利用AI的客觀性來規避人類參與者在特定情境下的社會期望偏見。然而，AI人格的生成仍依賴於訓練數據，若數據本身存在偏見，AI人格也可能繼承這些偏見，且其「真實性」仍是模擬而非實際體驗。
  • 傳統UX研究高度依賴人類參與者的直接回饋，但本研究指出這種回饋可能受環境影響。AI人格的引入提供了一種補充甚至替代方案，尤其是在需要規避社會偏見的敏感情境中，這代表了UX研究方法論的一大進展。
  • 這項研究可能徹底改變UX研究的執行方式，特別是在需要高度客觀性或難以接觸真實用戶的領域。它促使研究者重新思考「真實」數據的定義，並探索AI在設計思考流程中的更深層次應用。
研究貢獻
  • 提出AI人格能克服人類情境偏見的「偏見悖論」
  • 分享了使用OpenAI GPT建構器開發AI人格的實踐經驗
  • 為UX研究提供了一種新的、更客觀的數據獲取方法
研究限制
  • AI人格的生成仍可能受訓練數據偏見影響
  • AI人格無法完全替代人類在情感與細微體驗上的洞察

No. 03
Nothing from Something: Can a Language Model Discover 0?
cs.AIcs.CL 2026-06-18
基於人工神經網路的AI系統正被開發，旨在突破人類數學知識的邊界。這些系統能否超越其訓練數據進行發現，是一個關鍵問題。數學發現需要一種強大的分佈外泛化能力，即假設真正新的、可能邏輯上更強大的數學結構的能力。有假說認為語言能力支持人類認知中的此類泛化。本研究使用簡單的語言模型來探討其是否能發現「0」這個概念。實驗結果顯示，在特定條件下，語言模型展現出從現有數字系統中推導出「0」的潛力。這項工作為理解AI的數學創造力與泛化能力提供了新的視角。
語言模型能否自主發現像「0」這樣的新數學概念？
深度點評
  • 研究透過設計巧妙的實驗，探討LLM在有限數字系統中推導「0」的能力，這是一種新穎的探究AI數學發現潛力的方法。然而，實驗的簡化環境可能無法完全反映真實數學發現的複雜性，且「發現」的定義仍需進一步釐清。
  • 傳統上認為數學發現是人類獨有的高級認知能力。本研究挑戰了這一觀點，展示了LLM在特定條件下超越訓練數據進行概念推導的可能性，這對AI的創造性與泛化能力研究是一個重要進展。
  • 這項研究不僅深化了我們對AI認知能力的理解，也為開發能進行真正數學發現的AI系統奠定了基礎。它可能促使認知科學家重新審視語言與數學思維之間的關係。
研究貢獻
  • 探討了語言模型發現抽象數學概念「0」的潛力
  • 設計了評估LLM分佈外泛化能力的實驗框架
  • 為理解AI的數學創造力提供了實證依據
研究限制
  • 實驗環境的簡化性可能無法完全模擬真實數學發現
  • 對「發現」的定義與衡量仍有討論空間

Talking to Your Data: Exploring Embodied Conversation as an Interface for Personal Health Reflection
cs.HCcs.AI 2026-06-18
來自穿戴裝置的個人健康數據通常以圖表和統計摘要的形式呈現，要求用戶主動解釋模式和含義。本研究探索了一種替代的互動範式：透過具身對話代理與個人健康數據互動，以對話方式促進用戶客觀地反思數據。我們提出了一個系統，結合了穿戴數據的輕量級預處理與基於Unity的具身角色。在內部，該系統遵循雙代理設計，其中一個代理負責數據分析，另一個代理負責對話生成。用戶可以與這個虛擬代理進行自然語言對話，詢問關於其健康數據的問題，並獲得個性化的見解和建議。這項研究展示了對話式AI在個人健康管理中的巨大潛力。
具身對話代理如何作為個人健康數據反思的介面？
深度點評
  • 該研究創新性地將具身對話代理引入個人健康數據反思，透過自然語言互動降低了數據解讀的認知負荷。然而，代理的「具身」效果及其對用戶信任和依從性的影響，仍需更深入的心理學評估。
  • 相較於傳統的健康儀表板，本系統提供了更直觀、個性化且互動性強的數據呈現方式，有望提升用戶對健康數據的理解與參與度，是HCI與健康心理學領域的重要進展。
  • 這項技術可能改變人們與個人健康數據互動的方式，從被動查看轉為主動對話與反思。它為開發更具人性化和支持性的數位健康工具提供了藍圖，並可能促進健康行為的改變。
研究貢獻
  • 提出具身對話代理作為個人健康數據反思的互動介面
  • 開發了結合數據預處理與雙代理設計的系統
  • 探索了對話式AI在個人健康管理中的應用潛力
研究限制
  • 具身代理對用戶心理影響的深入評估不足
  • 系統對複雜健康數據解釋的準確性與安全性仍需驗證

No. 05
Quantifying Consistency in LLM Logical Reasoning via Structural Uncertainty
cs.AIcs.CL 2026-06-18
大型語言模型在多步驟演繹推理中，即使能得出相同答案，其推理路徑也可能不穩定、矛盾或難以一致地排序。現有方法主要透過輸出分散性來評估可靠性，即測量採樣答案的差異，但這忽略了一個互補的信號：模型能否一致地排序競爭性推理候選。本研究提出「結構不確定性」，這是一個基於自我一致性穩定性導出的、意識到一致性的框架。該框架能評估模型在不同推理路徑間的穩定性，從而更全面地量化LLM邏輯推理的一致性。實驗結果表明，結構不確定性在識別LLM推理缺陷方面比傳統方法更有效。
如何量化LLM邏輯推理路徑的一致性？
深度點評
  • 本研究提出的「結構不確定性」框架，透過評估模型對不同推理候選的排序穩定性來量化一致性，這比單純的輸出分散性更具洞察力。然而，該方法可能需要多次採樣和比較，計算成本較高，且其對「一致性」的定義仍可能受限於模型內部表示。
  • 現有方法多關注LLM推理結果的正確性或多樣性，而忽略了推理過程本身的一致性。本研究填補了這一空白，為評估LLM的「思維」可靠性提供了更精細的工具，是LLM可解釋性與可靠性研究的重要進展。
  • 提升了對LLM推理可靠性的理解，特別是在需要高精度和可信度的應用場景（如法律、醫療決策）中，這項研究有助於開發更穩健、更值得信賴的AI系統。它也為認知科學研究提供了新的工具，以探討人類與AI推理的異同。
研究貢獻
  • 提出「結構不確定性」框架以量化LLM邏輯推理的一致性
  • 強調了推理路徑穩定性作為評估LLM可靠性的重要信號
  • 開發了比傳統方法更有效的推理缺陷識別工具
研究限制
  • 計算成本可能較高，尤其是在複雜推理任務中
  • 對「一致性」的衡量仍可能受限於模型內部表示的透明度

04 / Prompt Engineering × 心理學 精選 TOP 3 PROMPT ENG × PSYCHOLOGY
Prompt × 心理學 No.01
Beyond Parallel Sampling: Diverse Query Initialization for Agentic Search
cs.AI
Prompt 技術
Diverse Query Initialization (DivInit)。這是一種訓練無關的干預措施，旨在解決代理式搜索中標準並行採樣導致的查詢冗餘問題。它透過在第一輪生成多樣化的初始查詢，確保不同搜索路徑能探索更廣泛的證據空間，從而提高搜索效率和結果質量。
心理學概念
認知搜索策略與冗餘。人類在解決問題時，若初始策略過於單一，容易陷入局部最優解。DivInit透過強制多樣化的初始查詢，模擬了人類在探索未知領域時，主動發散思維、避免「確認偏誤」的認知策略，旨在克服因初始信息重疊導致的「群體思維」效應。
應用場景

Prompt 技術
Pseudo Prompting Language (PromptMN)。這是一種領域特定語言，用於註釋自然語言提示，明確定義角色、目標、約束和預期輸出。它將隱含的提示信息顯性化，以減少上下文歧義，從而提高生成式AI在代理和軟體開發工作流程中的可靠性。
心理學概念
認知負荷與溝通清晰度。人類在處理模糊或不完整的指令時，會產生較高的認知負荷，且容易產生誤解。PromptMN透過結構化提示，降低了AI模型對指令的解釋難度，減少了「語義模糊」帶來的認知摩擦，類似於人類溝通中強調「清晰、簡潔、具體」的原則，以確保信息準確傳達。
應用場景
在需要精確控制AI行為的場景，如自動化程式碼生成、複雜任務規劃或多代理協作中，PromptMN能大幅減少因提示歧義導致的錯誤。它使得開發者能更有效地與AI溝通，確保AI行為符合預期。對心理學實務而言，這啟發我們在設計心理干預或教育材料時，應盡可能使用清晰、無歧義的語言和結構，以降低受眾的認知負荷，提高信息傳達的有效性。
