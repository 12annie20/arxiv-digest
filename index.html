State Representation and Termination for Recursive Reasoning Systems
cs.AI 2026-05-11
大型語言模型在複雜推理任務中面臨挑戰，尤其是在需要多步驟遞歸思考時。本研究旨在解決遞歸推理系統中狀態表示與終止條件的隱性設計問題。作者提出將推理狀態表示為一個包含主張、證據關係、開放問題和置信權重的「認知狀態圖」。研究定義了「順序差距」來衡量不同推理順序（擴展後整合 vs. 整合後擴展）之間的狀態距離，並利用此差距作為終止迭代的標準。實驗結果顯示，這種明確的狀態表示和終止機制能有效提升推理系統的效率與準確性。這項工作為開發更穩健、可解釋的AI推理系統提供了理論與實踐基礎。
如何有效表示遞歸推理狀態並決定何時終止？
深度點評
  • 本研究創新性地將推理狀態建模為認知狀態圖，並引入「順序差距」作為終止條件，為遞歸推理系統提供了清晰的理論框架與實踐方法，超越了傳統隱式設計的局限。
  • 相較於僅依賴行為輸出的現有推理系統，本文透過內部狀態的明確表示，使得AI的推理過程更具可解釋性與可控性，有助於診斷錯誤並提升系統可靠性。
  • 這項工作對開發能夠進行複雜、多步驟推理的AI系統具有深遠影響，特別是在需要高透明度和可靠性的領域，如科學發現或法律分析。
研究貢獻
  • 提出以認知狀態圖表示遞歸推理狀態的框架
  • 定義「順序差距」作為衡量推理進程與終止的標準
  • 為AI推理系統的設計提供了更明確的狀態管理與終止機制
研究限制
  • 認知狀態圖的構建與維護可能在極大規模或高度動態的知識領域中面臨複雜性挑戰
  • 「順序差距」的閾值設定可能需要針對不同任務進行細緻調整，缺乏普適性

LLM代理的記憶機制如何從儲存演化為經驗學習？
深度點評
  • 本調查首次提出將LLM代理記憶機制置於「從儲存到經驗」的演化框架中，成功整合了計算機科學與認知科學的視角，為該領域提供了急需的統一理論基礎。
  • 與傳統僅關注記憶容量或檢索效率的研究不同，本文強調記憶在代理學習、適應與環境互動中的動態角色，深化了對LLM代理認知能力的理解。
  • 這項工作不僅是理解當前LLM代理記憶發展的里程碑，也為未來設計更具人類智能特徵、能從經驗中持續學習的AI代理提供了關鍵指導。
研究貢獻
  • 提出LLM代理記憶機制的演化框架
  • 彌合了計算機科學與認知科學在記憶研究上的鴻溝
  • 系統性回顧並分類了現有的LLM代理記憶架構
研究限制

如何利用AI輔導系統促進學生從表面學習轉向深度理解？
深度點評
  • 本研究透過RAG技術將LLM回應「紮根」於教師材料，有效解決了LLM在教育應用中常見的幻覺問題，顯著提升了輔導內容的可靠性與準確性。
  • 相較於傳統的自動化輔導系統，該插件引入了蘇格拉底式互動，鼓勵學生主動思考而非被動接收資訊，更符合深度學習的認知心理學原則。
  • 這項工作為教育科技領域提供了實用的AI輔導解決方案，特別是在Moodle等主流學習管理系統中，有望大幅提升教學效率與學習成果，同時減輕教師負擔。
研究貢獻
  • 開發基於RAG的Moodle AI輔導插件，解決LLM幻覺問題
  • 實現蘇格拉底式互動輔導，促進深度學習
  • 提供教師「人機協作」介面，確保內容品質與控制
研究限制
  • 系統的有效性高度依賴教師提供材料的品質與完整性

研究貢獻
  • 提出「以人為本的大型語言模型」（HCLLMs）的綜合框架
  • 整合NLP、HCI和社會科學視角，強調用戶需求與倫理考量

LLM × 心理學 No.01
When Does a Language Model Commit? A Finite-Answer Theory of Pre-Verbalization Commitment
cs.CLcs.AI
核心問題
語言模型何時在生成最終答案前形成穩定偏好？
方法亮點
本研究透過「有限答案偏好穩定化」來探討語言模型在生成推理過程中的承諾點。對於給定的模型狀態和答案詞彙化器，研究將模型的連續機率投射到有限答案集上，並在二元任務中計算精確的對數賠率碼 $\delta(\xi)$。這個目標定義了一個解析器，用於追蹤模型在生成最終答案之前，其答案偏好何時變得穩定。
心理學意涵
這項研究深入探討了LLM的決策過程，揭示了模型在「思考」過程中形成「承諾」的認知機制。這對理解AI的「心智」運作方式，特別是其內部推理與外部表達之間的關係，具有重要意義。在心理學上，這類似於人類在做出決定前，其內部信念或偏好如何逐步穩定。這有助於提升AI的可解釋性，並可能啟發對人類決策過程的計算模型研究。同時，理解AI的承諾點也能幫助設計更可靠、更值得信賴的AI系統，尤其是在高風險決策場景中。
揭示LLM「心智」的承諾時刻，為AI決策透明度與人類認知理解開啟新篇章。

LLM × 心理學 No.02
Domain-level metacognitive monitoring in frontier LLMs: A 33-model atlas
cs.AIcs.CL
核心問題
頂尖LLM的元認知監控能力在不同領域間是否存在差異？
方法亮點
研究對33個頂尖LLM進行了1,500個MMLU項目測試（每個領域250個，分為六個預設領域），並使用口頭表達的置信度（0-100）計算每個模型-領域單元的Type-2 AUROC。總共收集了47,151個觀測值，以量化模型在不同知識領域的元認知監控品質。
心理學意涵
本研究首次大規模揭示了LLM在不同知識領域的元認知監控能力差異，這對理解AI的「自我意識」和「認知限制」具有深遠的心理學意義。它表明LLM並非在所有任務上都同等了解自己的能力邊界，這與人類在不同專業領域的元認知表現有相似之處。這項發現有助於改進AI的校準和信任度，特別是在需要AI提供置信度評估的應用中。同時，它也為認知心理學研究提供了新的視角，可以將LLM作為一個可控的實驗對象來探索元認知現象。
LLM的元認知能力並非一視同仁，領域差異揭示AI「自知之明」的複雜性。

IntentGrasp: A Comprehensive Benchmark for Intent Understanding
cs.CLcs.AI
核心問題
如何全面評估LLM的意圖理解能力？
方法亮點

More Thinking, More Bias: Length-Driven Position Bias in Reasoning Models
cs.AIcs.CL
Prompt 技術
Chain-of-Thought (CoT) Reasoning。CoT透過引導模型生成中間推理步驟，以提升其在複雜任務上的表現。本研究探討了CoT推理長度與模型偏見之間的關係。
心理學概念
認知偏見 (Cognitive Bias) 與推理 (Reasoning)。研究發現，CoT推理的長度與模型在多選題中的位置偏見呈正相關，挑戰了CoT能減少淺層啟發式偏見的普遍假設。這表明即使AI進行了「更多思考」，也可能加劇而非減輕某些認知偏見。
應用場景
當前，CoT廣泛應用於提升LLM的推理能力，但本研究提醒我們，在設計CoT提示時需警惕其可能引入或加劇的偏見。未來，在教育輔導、決策支持等需要高公平性和準確性的AI應用中，應對CoT生成的推理過程進行更嚴格的偏見審查。對心理學實務而言，這啟發我們在分析人類推理時，也要考慮「思考過程長度」與偏見之間的潛在關聯，並設計相應的去偏見策略。
CoT越長，偏見越深，AI「思考」的副作用不容忽視。

Prompt × 心理學 No.02
When Does Critique Improve AI-Assisted Theoretical Physics? SCALAR: Structured Critic--Actor Loop for Agentic Reasoning
cs.AIcs.HC
Prompt 技術
Critic-Actor Loop (評論者-行動者循環)。這是一種代理式推理框架，其中「行動者」提出解決方案，「評論者」提供迭代反饋，而獨立的「評審者」則根據參考答案評估整個過程。這種循環機制類似於人類的自我反思與同行評審。
心理學概念
反饋 (Feedback)、學習 (Learning)、批判性思維 (Critical Thinking) 與人機協作 (Human-AI Collaboration)。評論者-行動者循環模仿了人類學習和改進的過程，即透過接收外部或內部反饋來修正行為。這與心理學中關於學習理論和認知改進的機制高度相關，尤其是在複雜問題解決中，反饋的質量和結構對學習效果至關重要。
應用場景
SCALAR框架在AI輔助科學研究中展現了巨大潛力，特別是在理論物理等需要嚴謹推理的領域。它能提升AI解決問題的準確性和可靠性，同時優化人機協作模式。未來，這種帶有結構化批判的反饋機制可應用於AI輔助設計、程式碼審查、醫療診斷等領域，幫助人類專家與AI共同提升決策品質。對心理學實務而言，這啟發我們如何設計更有效的反饋機制，以促進人類在複雜任務中的學習與表現。
AI的「自我批判」與人類協作，共同推動科學發現的邊界。

Prompt × 心理學 No.03
Can LLMs Take Retrieved Information with a Grain of Salt?
cs.AIcs.CL
Prompt 技術
Retrieval-Augmented Generation (RAG)。RAG透過從外部知識庫檢索相關資訊來增強LLM的生成能力，旨在減少幻覺並提高事實準確性。本研究探討了LLM對檢索資訊「確定性」的適應能力。
心理學概念
認知確定性 (Cognitive Certainty)、知識整合 (Knowledge Integration) 與信任 (Trust)。研究發現LLM在根據檢索資訊的確定性調整回應方面存在系統性限制，即使明確告知資訊不確定，模型仍難以回憶先驗知識。這反映了LLM在處理不確定性資訊和整合多源知識時的認知挑戰，類似於人類在面對模糊或矛盾資訊時的認知偏差。
應用場景
RAG技術已廣泛應用於問答系統、內容生成等領域，但本研究揭示了其在處理資訊確定性方面的潛在風險。在高風險領域（如醫療、金融），AI需要精確傳達資訊的確定性，以避免誤導用戶。未來RAG系統需進一步優化，使其能更精準地反映資訊的不確定性。對心理學實務而言，這提醒我們在設計人機互動時，應考慮AI如何傳達其「知識邊界」，以建立更健康的用戶信任關係，並避免用戶過度依賴AI的「確定性」輸出。
RAG雖強，但LLM對資訊「確定性」的理解仍需打磨，信任建立任重道遠。
