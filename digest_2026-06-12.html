No. 01
Position: Hippocampal Explicit Memory Is the Cornerstone for AGI
cs.AI 2026-06-12
這篇立場論文主張，將顯性記憶整合到大型語言模型（LLMs）中，是實現通用人工智慧（AGI）的基石。LLMs的學習機制與人類內隱記憶高度相似，但AGI所需的高階認知功能，如長期策略規劃、後設認知和符號推理，卻嚴重依賴海馬迴的顯性記憶。研究指出，顯性記憶能提供可追溯的知識、支持因果推理和情境理解，這些都是LLMs目前難以有效處理的。因此，作者認為，若要讓LLMs超越當前的能力限制，必須開發能有效模擬和利用顯性記憶的架構與機制。這不僅能提升模型的透明度和可解釋性，也能使其在複雜任務中展現更接近人類的智慧。
顯性記憶如何成為AGI發展的關鍵？
深度點評
  • 本文為立場論文，主要在概念層面進行論證，而非提出具體模型或實驗。其創新在於明確指出顯性記憶在AGI中的核心地位，挑戰了當前LLM過度依賴隱性學習的趨勢。
  • 既有研究多專注於提升LLM的參數規模或訓練數據，以增強其隱性知識。本文則從認知科學角度，提出整合生物學啟發的顯性記憶機制，為AGI發展提供了一個全新的、更具解釋性的方向。
  • 本文可能引導AI研究者重新思考AGI的架構，鼓勵開發結合符號推理與神經網路的混合模型，並促進AI與認知神經科學的跨領域合作，以實現更具人類智慧的AI系統。
研究貢獻
  • 明確提出顯性記憶對AGI的關鍵作用。
  • 將LLM的學習機制與人類內隱記憶進行類比。
  • 強調高階認知功能對顯性記憶的依賴。
研究限制
  • 缺乏具體的模型架構或實驗驗證。
  • 主要為理論性論證，實踐路徑尚不明確。

No. 02
Knowing When to Ask: Self-Gated Clarification for Hierarchical Language Agents
cs.AIcs.CL 2026-06-12
在階層式推理中，AI代理常因在中間決策點缺乏關鍵資訊而做出錯誤承諾。本研究提出ACTION-RATING框架，將澄清（asking）納入代理的行動空間，使其與導航（acting）在同一序數尺度上競爭。這使得代理在每個決策點都能評估是繼續行動還是尋求幫助，從而使求助行為在中間狀態變得可觀察。研究引入兩種結構不同的資訊尋求機制，並透過實驗證明此框架能有效提升代理在複雜任務中的決策品質。這種內建的自我門控澄清機制，有助於代理更智能地識別自身知識邊界，減少錯誤累積，並提高推理過程的透明度。
如何讓階層式語言代理知道何時尋求澄清？
深度點評
  • 創新之處在於將「尋求澄清」內化為代理的行動選項，而非外部觸發機制，並使其與其他行動在同一尺度上競爭，這模擬了人類在決策時的元認知過程。
  • 傳統方法多將澄清視為外部不確定性觸發，或依賴預設規則。本研究則透過將澄清納入代理的行動空間，實現了更自主、更靈活的資訊尋求策略，顯著提升了代理在複雜推理任務中的魯棒性與準確性。
  • 此研究為開發更智能、更具人類認知能力的AI代理提供了新思路，特別是在需要多步驟推理和不確定性處理的場景。它也為人機協作中AI代理的「提問」行為提供了理論基礎和實踐指導。
研究貢獻
  • 提出ACTION-RATING框架，將澄清內化為代理行動。
  • 使代理在中間決策點能自主評估是否尋求幫助。
  • 引入兩種資訊尋求機制並驗證其有效性。
研究限制
  • ACTION-RATING評分機制可能需要精細調校。
  • 在極端複雜或開放域任務中的表現仍待進一步驗證。

No. 03
Understanding and Supporting Online Discussion with Opinionated Chatbots
cs.HCcs.CY 2026-06-12
帶有觀點的聊天機器人日益普及，可能透過影響個人觀點來塑造公共論述。本研究探討與不同類型觀點聊天機器人互動後，對參與者後續線上討論的影響。透過一項受控實驗，參與者接觸表達相反、強化或平衡觀點的聊天機器人，然後參與線上討論。研究發現，聊天機器人的觀點類型顯著影響了參與者的討論行為、觀點表達和對議題的理解深度。結果顯示，平衡觀點的聊天機器人有助於促進更開放和多元的討論，而強化觀點的機器人可能導致觀點極化。這項研究為設計支持健康線上公共論壇的AI工具提供了重要見解。
觀點聊天機器人如何影響線上討論與公共論述？
深度點評
  • 創新在於透過受控實驗，系統性地比較了不同觀點類型聊天機器人對線上討論的影響，提供了量化和質化的證據。侷限可能在於實驗環境與真實社交媒體環境的差異，以及參與者樣本的代表性。
  • 既有研究多關注聊天機器人的說服力或資訊傳播，本研究則進一步探討其對「後續人際互動」和「群體論述形成」的影響，填補了AI在社會心理學領域的空白。
  • 本研究對設計負責任的AI聊天機器人、維護線上公共論壇的健康生態具有重要指導意義。它提醒開發者需考慮AI工具的社會心理影響，並鼓勵設計能促進多元對話而非觀點極化的AI。
研究貢獻
  • 首次系統性研究觀點聊天機器人對後續線上討論的影響。
  • 區分相反、強化、平衡三種觀點類型。
  • 提供設計支持健康公共論述AI工具的實證依據。
研究限制
  • 實驗環境與真實社交媒體的複雜性存在差異。
  • 長期影響和跨文化差異未被探討。

Frozen Multimodal Embeddings for Personality and Cognitive Ability Assessment in Asynchronous Video Interviews
cs.AIcs.HC 2026-06-12
從非同步視訊面試（AVIs）預測心理特質是一個挑戰性的多模態學習問題，因為標註數據有限，且每個回應都包含高維度的視覺、聽覺和語言訊號。本論文提出了解決ACM多媒體AVI挑戰賽2026的方案，該方案評估兩個任務：任務1從與個性相關的面試回應中預測自我報告的HEXACO人格特質，任務2則從結構化的AVI回應中分類認知能力水平。我們將此問題視為一個少樣本學習任務，並利用預訓練的多模態嵌入（如CLIP、Wav2Vec 2.0和BERT）作為凍結特徵提取器。透過輕量級的下游分類器，本方法在有限數據下實現了穩健的心理特質評估。
如何從非同步視訊面試中有效評估人格與認知能力？
深度點評
  • 創新在於利用凍結的預訓練多模態嵌入作為特徵提取器，有效解決了心理學數據集標註困難和數據量有限的問題，降低了模型訓練的複雜性。
  • 傳統方法常需大量標註數據或依賴單一模態。本研究透過多模態融合和遷移學習，在數據稀缺的心理學評估領域取得了顯著進展，提供了一種更高效、更穩健的解決方案。
  • 此研究為心理學評估，特別是人力資源篩選、臨床診斷輔助等領域，提供了自動化、標準化的新工具。它有助於提高評估效率和客觀性，但也引發了關於AI評估公平性和倫理的討論。
研究貢獻
  • 提出基於凍結多模態嵌入的心理特質評估方案。
  • 有效解決非同步視訊面試數據稀缺問題。
  • 在人格特質和認知能力評估任務中表現出色。
研究限制
  • 預訓練模型的潛在偏見可能影響評估公平性。
  • 對不同文化背景下的適用性需進一步驗證。

No. 05
Identifying cybersickness causes in virtual reality games using symbolic machine learning algorithms
cs.HC 2026-06-12
虛擬實境（VR）和頭戴式顯示器在教育、軍事、娛樂和健康等領域日益普及。儘管這些技術提供了高度沉浸感，但也可能引發不適症狀，即所謂的「暈動症」（cybersickness, CS）。本研究提出一種新穎的實驗分析方法，利用符號機器學習演算法來排序VR遊戲中CS的潛在原因。透過估計CS原因並根據其影響力進行排序，研究旨在識別導致不適的核心因素。這項工作有助於開發更舒適的VR體驗，並為VR內容設計者提供具體的指導，以減少用戶的暈動症。
如何識別並排序虛擬實境暈動症的潛在原因？
深度點評
  • 創新在於採用符號機器學習演算法來分析和排序暈動症原因，這提供了比傳統統計方法更具解釋性和可操作性的結果。侷限可能在於符號模型在處理高維度、非線性數據時的表現，以及其對數據品質的依賴。
  • 既有研究多集中於暈動症的測量或預防，但對其「根本原因」的識別和排序較少深入。本研究透過機器學習方法，為理解暈動症的機制提供了新的工具和視角。
  • 本研究對VR遊戲和應用開發者具有直接的實踐價值，可指導他們在設計時避免或減輕導致暈動症的因素，從而提升用戶體驗和VR技術的普及率。
研究貢獻
  • 提出利用符號機器學習識別VR暈動症原因。
  • 對暈動症潛在原因進行影響力排序。
  • 為開發更舒適VR體驗提供具體指導。
研究限制
  • 符號機器學習模型的解釋力可能受限於特徵工程。
  • 實驗設計和數據收集的全面性可能影響結果的泛化。

Benchmarking Large Language Models for Safety Data Extraction
cs.CLcs.AI
Prompt 技術

心理學概念
認知負荷 (Cognitive Load)、推理 (Reasoning)、偏見 (Bias)。零樣本提示對模型認知負荷最低，但可能因缺乏上下文而表現不佳。少樣本和思維鏈提示透過提供結構化資訊或推理步驟，降低了模型的認知負荷，並引導其進行更複雜的推理。同時，提示的設計也可能引入或減輕模型在安全數據提取中的潛在偏見。
應用場景
當前應用於工業安全領域的安全數據表（SDS）資訊提取，可自動化處理異質文件格式，提高數據提取的準確性和效率。未來潛力在於將此技術推廣到其他需要精確資訊提取的領域，如醫療記錄、法律文件等。對心理學實務的啟發是，提示工程可作為一種工具，用於設計更有效的人機協作流程，例如輔助心理諮詢師從大量文本中提取關鍵資訊，或用於開發更具解釋性的AI輔助診斷系統。
提示策略巧運用，安全數據提取不再難，AI推理透明度大躍進。

自我門控澄清 (Self-Gated Clarification) / 行動評級 (ACTION-RATING)。這是一種將「尋求澄清」作為代理內部行動空間一部分的設計，代理在每個決策點評估是繼續行動還是尋求外部幫助，並將其與其他行動在同一序數尺度上競爭。
心理學概念
元認知 (Metacognition)、決策 (Decision-making)、認知控制 (Cognitive Control)。自我門控澄清機制模擬了人類的元認知能力，即「知道自己不知道」並主動尋求資訊。代理在決策過程中展現了認知控制，權衡行動與求助的成本與收益。這有助於減少代理在不確定性下的錯誤承諾，提升推理的效率和準確性。
應用場景
當前應用於階層式推理任務中，提升語言代理在複雜決策點的表現。未來潛力在於開發更具自主學習和適應能力的AI代理，使其在開放世界環境中能更智能地與人類協作，例如在複雜的設計、研究或問題解決任務中。對心理學實務的啟發是，這種機制可以應用於設計輔助人類決策的AI系統，使其在提供建議時能更智能地識別自身知識邊界，並在必要時主動向人類尋求澄清，從而建立更高層次的人機信任。
AI學會「自問自答」，決策更聰明，元認知能力再升級。

NightFeats @ MMU-RAGent NeurIPS 2025: A Context-Optimized Multi-Agent RAG System for the Text-to-Text Track
cs.AIcs.CL
Prompt 技術
代理式上下文工程 (Agentic Context Engineering, ACE) / 多代理檢索增強生成 (Multi-Agent RAG)。ACE是一種為代理設計優化上下文的策略，透過將知識合成分解為檢索、策劃和組合三個階段，每個階段由明確的中間表示和交接契約管理。多代理RAG系統則利用多個協調的代理來執行這些階段。
心理學概念
認知負荷 (Cognitive Load)、資訊處理 (Information Processing)、協作 (Collaboration)。ACE和多代理RAG透過結構化任務和上下文管理，有效降低了單一模型處理複雜資訊的認知負荷。將知識合成分解為多個階段，並由專業代理處理，模擬了人類在複雜任務中分工協作和資訊整合的過程，提升了資訊處理的效率和準確性。
應用場景
當前應用於文本到文本的檢索增強生成任務，特別是在需要複雜知識合成的場景。未來潛力在於開發更高效、更可解釋的AI研究助理或知識管理系統，能從海量資訊中提取、整合並生成高質量的結論。對心理學實務的啟發是，這種多代理協作模式可以應用於輔助心理學研究者進行文獻回顧、數據分析和報告撰寫，透過結構化的資訊處理流程，減少研究者的認知負擔，提高研究效率。
多代理RAG系統，知識合成新典範，AI協作智慧再進化。
